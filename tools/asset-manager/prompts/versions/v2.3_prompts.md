# v2.3 Phase A — B4.3 完整 Prompt 归档

- 实验日期：2026-04-18
- 前置基线：v2.2 的 **B4**（cinematic / painterly / warm_signal mean = 0.156）
- 本轮目的（**极小改动，只改两处**）：
  1. **统一背景信号为 transparent**：B4 的 `style_base` 写的是 `against pure white background`，而 `portrait_template` 里写的是 `Transparent background, nothing else behind the character`，两者冲突 → 抠图到游戏内时边缘残留不干净。B4.3 只把 `style_base` 里的 `pure white background` 改为 `clean transparent background`，保持整张 prompt 只传递一个背景信号。
  2. **加入极度克制的时代锚点**：B4 缺少朝代锚点 → 服饰偏"泛东亚"。B4.3 在 `portrait_template` 的 `Subject` 行之后、`pose` 行之前，**插入一行短句**，只包含 1–2 条锚点：`Tang-Song era jianghu attire with layered robes and woven sashes where appropriate.`。
- description 层：**复用 v1_20260418**（`--descriptions-from v1_20260418`），严格单因子。
- 生产文件触碰：**0**（只读模式，templates 通过 `--templates-from` 注入）。
- **其他所有区域一字不动**：`no_text_constraint` / `style_negative` / `item_template` / `scene_template` 全部与 B4 byte-for-byte 相同；`portrait_template` 除新增一行外其他部分保持原状。

### 5.2 肉眼判断

- **透明底**：✅ 两个 run 的透明像素占比都在 55–71% 范围，背景层干净可抠。冲突信号的移除本身没有引入 pixel-level 故障（alpha 通道正常），主要的价值是 **prompt 层消除歧义**。
- **唐宋锚点**：✅ 到位，没有过度化。
  - 温华 v1：原 B4 是"泛东亚贫民青衫"；B4.3 是"分层交领短衣 + 编织腰带 + 内衬可见"，明显朝唐宋平民武人方向收敛，但没有工笔 / 玉佩 / 刺绣。
  - 老黄 v1：原 B4 是单层粗衣；B4.3 出现 **双层外衣 + 斜挎剑匣带**，典型唐宋布衣老仆形态。
  - 姜泥 v1：原 B4 是单色麻衣；B4.3 出现 **内外两层（外袍 + 内裳） + 宽腰带**，经典唐宋女性层叠衣着。
- **过度化风险**：❌ 未触发。没有 embroidery / jade / gongbi silk（constraint 尊重）。
- **暖色代价**：⚠️ 模型把"Tang-Song era"自由解读为 **土黄 / 赭石 / 棕 / 铁锈色** 的时代质感，导致整体 warm_signal 回潮。老黄 v2 是典型案例（外袍整体偏棕）。这是 **锚点自带的时代色偏**，不是 bug，但需要在下一版考虑是否用冷色 anchor（如 "Tang-Song era jianghu attire **in muted indigo/slate palette**"）来抵消。

#### (b) 是否有副作用／过拟合

- **没有出现"老黄土黄化"那种全局过拟合**：warm_signal 全局 Δ 只有 +0.0001，std 略微放大（+0.0064），说明 B4.3 不是"整体变黄"，而是**按角色分化**。
- **真正的过拟合不在颜色，而在造型约束**（这是本轮新样本暴露的新风险）：`layered robes and woven sashes where appropriate` 里的 `where appropriate` 并没有完全阻止模型对"非层叠造型"也强加层叠——**陈芝豹的银甲 + 褚禄山 v1 的皮甲** 在 B4.3 都被 Tang-Song 服饰协议改写，**identity 漂移**比"warm_signal 偏高"更难挽回。
- **色偏副作用仍在但分化**：温华 (+0.0243) / 老黄 (+0.0215) / 南宫 (+0.0229) 是"布衣 / 粗袍 / 素裙"档的角色，B4.3 会把他们的面料推向**棕 / 土黄 / 米色**（模型对 Tang-Song 的联想），这类 warm 漂移是"时代色偏"的典型；李淳罡 (−0.0169) / 轩辕 (−0.0169) / 褚禄山 (−0.0312) 都有**明确色彩设定（青灰道袍 / 紫袍 / 锈红甲胄）**，B4.3 反而把颜色**压得更深饱和更冷**，因此 Δ 为负。

### 6.5 最终建议：B4 vs B4.3 各自适合的角色类型（基于扩样本）

> 美学选择权在用户，以下仅给出基于 36 张图 + 肉眼观察的客观分组建议，不做选型决策。

**B4.3 优势角色（朝代层叠带来净增益、warm 漂移可接受）：**
- 布衣 / 素衣 / 平民 江湖人：温华、姜泥、老黄（日常）、南宫仆射
- 强色彩设定但需要服装层次的：轩辕青锋
- 便服 / 闲居 场景：褚禄山 v2 类

**B4 优势角色（B4.3 会触发 identity 漂移或削弱标志性造型）：**
- 甲胄主导 / 非层叠造型：陈芝豹（白衣银甲）、褚禄山 v1（皮甲悍匪）
- 剑意 / 仙气主导、不希望时代感太实：李淳罡（B4.3 虽然变冷是好事，但也吃掉了"青灰道袍"的仙气，Δ 是"副产物收益"不是"目标收益"）

**模糊带（两版差异小、各有取舍）：**
- 徐凤年：Δ mean = −0.0070，两版都能用，B4.3 的层叠可能让锦衣公子更贵气，B4 更干净；建议按 variant 决定
- 陈芝豹 v2：静态场景，两版差距较小

### 6.6 对原 §5 结论的校正

- 原 §5 在 5 老角色 × 4 variant 上得 B4.3 比 B4 整体 +0.0116 warm、**小幅倒退**。在本轮扩到 9 × 2 后，全局 Δ 几乎归零（+0.0001）。**"B4.3 整体偏暖"不成立，真实情况是"色偏按角色类型分化"**。
- 原 §5 认为"老黄 +0.032 最严重"。本轮 **variant 1/2 子集** 下温华 (+0.0243) 才是老角色中最严重的暖化者，老黄只 +0.0215（但 variant 3/4 贡献了额外暖化，合并起来老黄仍是重灾区——若后续需要 4 variant 全量对比可再跑）。
- 原 §5 建议的"B4.4 方案"本轮**已取消**，因为扩样本后发现真正的风险是 **identity 漂移（甲胄被层叠协议覆盖）**，这不是换一个色板就能解决的。如果下一轮还要做 template 优化，应该改成"锚点条件化"（遇甲胄造型时自动跳过 layered robes），而不是再改色。
