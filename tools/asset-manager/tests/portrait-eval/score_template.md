## 评分维度（1-5 或 P/F）

| 维度 | 1 分 | 3 分 | 5 分 |
|------|------|------|------|
| 色温（色偏/泛黄）     | 整张明显泛黄/泛暖 | 偶有轻微色偏     | 白底干净、色调中性 |
| 风格一致性           | 明显西化或 3D 感  | 风格游离但大致一致 | 工笔 + 水墨稳定统一 |
| 气质辨识度           | 像路人           | 约一半画面抓到气质 | 一眼认出角色定位 |
| 面部准确度           | 脸型明显变形     | 尚可辨认         | 脸型/五官符合原著 |
| 服饰匹配度           | 与 description 明显不符 | 部分细节对应 | 服饰、饰品、兵器完全对应 |
| 姿态表现力           | 僵硬呆站         | 常规站姿         | 有故事感 / 招式感 |
| 技术合格（P/F）      | F：缺脚/多指/裁切/出现文字/有背景 | — | P：无缺陷 |
