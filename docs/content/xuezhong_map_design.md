# 《雪中悍刀行》游戏大地图与场景规划设计

## 1. 设计目标

本方案用于给 `sutan` 项目的大地图系统提供世界分区、场景节点和解锁节奏参考。设计原则如下：

- **贴合原著世界观**：以北凉、离阳、北莽、南疆、武帝城、宗门江湖为核心空间。
- **兼顾叙事与玩法**：每个场景既承担剧情氛围，也承担卡牌策略、资源管理、角色招募、权谋周旋等功能。
- **适配现有系统**：场景类型优先落到现有 `event / challenge / shop` 三类结构中，便于后续拆成 JSON。
- **强调路线感**：每张地图都不是散点集合，而是有主干线、支线、补给点和高风险节点。

## 3. 地图一：北凉道与边塞

### 3.1 地图定位

- **对应区域**：北凉王府统辖范围，包含清凉山到幽州、流州、葫芦口的边线。
- **视觉主题**：荒寒北地、军镇、驿路、芦苇荡与烽火台并存。
- **叙事用途**：作为开局主地图，负责建立“北凉铁骑、徐家旧部、边塞民生、江湖初逢”四条基础体验。
- **推荐主路线**：清凉山王府 → 陵州城 → 白马游弩校场 → 幽州马市 → 流州边关 → 葫芦口前哨。

### 3.2 场景节点

1. **清凉山王府**  
   - 类型：主线 / 枢纽 / 议政  
   - 相邻：听潮亭、陵州城、北凉暗驿  
   - 这里是北凉主城与主界面政治中枢，适合承接每日议事、人物互动、主线分派。玩法上可作为“派系关系结算点”和地图入口，提供招募、情报整理、主线分支选择。

2. **听潮亭**  
   - 类型：探索 / 情报 / 角色剧情  
   - 相邻：清凉山王府、王府藏道  
   - 这里承载北凉秘档、旧史残卷和高手传闻，适合做文本探索与卡牌事件链。玩法上可掉落情报卡、标签卡，或解锁角色专属回忆、隐藏剧情与新地图线索。

3. **陵州城市集**  
   - 类型：商店 / 民生支线  
   - 相邻：清凉山王府、芦苇荡渡口、北凉暗驿  
   - 陵州是开局最稳定的补给点，适合放置兵器、药材、马具、酒肆、说书摊等功能内容。玩法上承担基础商店、低风险事件和声望换资源，帮助玩家熟悉交易与城市事件。

4. **芦苇荡渡口**  
   - 类型：主线事件 / 江湖邂逅  
   - 相邻：陵州城市集、龙睛郡牧场  
   - 芦苇荡适合承接温华、李淳罡、老黄相关事件链，是“北凉线通往江湖线”的情绪场景。玩法上以角色检定和关系标签为主，成功后可获得高质量同伴、师承或债务型事件。

5. **白马游弩校场**  
   - 类型：战斗 / 训练 / 军旅  
   - 相邻：清凉山王府、幽州马市、龙睛郡牧场  
   - 这里是北凉军阵与骑射训练的核心场景，适合突出“北凉铁骑”的独特气质。玩法上可做多角色战斗检定、编队试训、兵源强化，也可加入武将好感与部曲培养。

6. **龙睛郡牧场**  
   - 类型：探索 / 资源 / 坐骑  
   - 相邻：芦苇荡渡口、白马游弩校场、幽州马市  
   - 牧场承接马匹、军粮、牧民请求等边塞生计内容，能缓冲主线战争压力。玩法上可产出坐骑、补给、草料类资源，也能出现盗马、狼袭、游侠比试等动态事件。

7. **幽州马市**  
   - 类型：商店 / 交易 / 阵营互动  
   - 相邻：白马游弩校场、龙睛郡牧场、流州边关  
   - 幽州马市是边军、商旅、游侠和草原商人混杂之地，最适合表现北地粗粝的交易氛围。玩法上可做高级装备购买、稀有坐骑交换、黑市竞价和情报倒卖。

8. **北凉暗驿**  
   - 类型：功能 / 驿站 / 情报传递  
   - 相邻：清凉山王府、陵州城市集、流州边关  
   - 这是偏功能型节点，视觉上可做荒野驿站与密信暗格。玩法上承担快速旅行、任务刷新、委托结算与跨地图情报传书，让玩家建立“北凉的手伸得很远”的感受。

9. **流州边关**  
   - 类型：主线 / 战斗 / 防线经营  
   - 相邻：幽州马市、北凉暗驿、葫芦口前哨  
   - 流州边关是北凉线从“经营内政”过渡到“直面战争”的关键节点。玩法上可加入守城、粮草调度、武将轮驻、军心检定与敌情侦察，逐步把玩家推向大战氛围。

10. **葫芦口前哨**  
   - 类型：高危战斗 / 前线探索  
   - 相邻：流州边关、拒北城方向出口  
   - 葫芦口适合作为北凉地图末端的高压节点，代表真正的边境杀机与北莽威胁。玩法上可以出现限时突袭、精锐战、牺牲抉择和前线救援，并作为解锁北莽地图的硬门槛之一。

## 4. 地图二：离阳京畿与中原官道

### 4.1 地图定位

- **对应区域**：太安城、官道枢纽、宗室别院、漕运节点、学宫与朝堂外围。
- **视觉主题**：宫阙深红、夜雨官灯、檐角飞兽、卷宗与金印。
- **叙事用途**：专注表现庙堂权谋、皇权制衡、离阳宗室与士林关系。
- **推荐主路线**：太安城朱雀门 → 尚书台 → 钦天监 → 皇城内库 / 天牢 → 襄樊水驿。

### 4.2 场景节点

1. **太安城朱雀门**  
   - 类型：主线 / 门槛 / 政治事件  
   - 相邻：尚书台、梨园别苑、青鸟官驿  
   - 朱雀门是进入离阳核心权力场的第一重门面，适合表现“名望不够进不去，关系不够站不稳”。玩法上可作为声望门槛节点，决定玩家能否触发更高阶朝堂事件。

2. **尚书台**  
   - 类型：主线 / 权谋 / 文官斗争  
   - 相邻：太安城朱雀门、皇城内库、天牢  
   - 尚书台适合用来做派系投票、文书伪造、诬告与弹劾链条。玩法上强调社交、智慧、情报卡消耗，成功可换取政策支持、资金调拨和官员关系网。

3. **钦天监**  
   - 类型：探索 / 预兆 / 异象  
   - 相邻：尚书台、上阴学宫驿路  
   - 钦天监不仅有朝廷术士，也有星象与天命的政治利用空间。玩法上适合生成预言型事件、灾异牌、时运增益或“伪造天象”的高风险操作。

4. **皇城内库**  
   - 类型：资源 / 潜入 / 高风险收益  
   - 相邻：尚书台、天牢  
   - 内库可作为离阳地图的财富与证物中心，既是肥肉也是陷阱。玩法上可通过权谋合法调用、潜入盗取、贿赂挪用等方式获得大量金钱、珍宝或黑材料，但失败后代价极重。

5. **天牢**  
   - 类型：主线 / 营救 / 审讯  
   - 相邻：尚书台、皇城内库、靖安别院  
   - 天牢是很多人物命运转折点，特别适合放“救人、灭口、递话、换供词”类任务。玩法上结合潜行、关系、资源消耗，允许玩家保人、弃子或反向设局。

6. **青鸟官驿**  
   - 类型：功能 / 驿站 / 跨图传令  
   - 相邻：太安城朱雀门、襄樊水驿、上阴学宫驿路  
   - 官驿让离阳地图具备明显的交通感与调度感，也方便连接中原与江南。玩法上提供跨地图任务投递、加急情报、时限任务刷新，是政治线的节奏器。

7. **上阴学宫驿路**  
   - 类型：支线 / 士林 / 辩论  
   - 相邻：钦天监、青鸟官驿、上阴学宫  
   - 这条路线适合承接书生、儒士、策论和名望任务，让玩家接触“天下读书种子”。玩法上偏智慧、魅力与口才判定，可招揽谋士、刷士林声望，或引发读书人与权臣的冲突。

8. **上阴学宫**  
   - 类型：探索 / 招募 / 长线关系  
   - 相邻：上阴学宫驿路  
   - 学宫是谋士、先生和年轻才俊的重要来源，适合做多轮拜访与名士事件。玩法上可以解锁特殊顾问卡、策略增益卡和“策论换资源”的中后期玩法。

9. **靖安别院**  
   - 类型：权谋 / 宗室 / 宴会  
   - 相邻：天牢、襄樊水驿  
   - 别院比皇城更适合做“笑里藏刀”的宴席事件，放大宗室、藩王、旧党之间的私人较量。玩法上可通过宴饮、送礼、诗会、暗杀等多种方式解决冲突，分支空间大。

10. **襄樊水驿**  
   - 类型：商路 / 军政 / 衔接江南  
   - 相邻：青鸟官驿、靖安别院、广陵江渡口  
   - 水驿是离阳地图通往江南与西楚故地的外沿出口，也是军粮漕运、盐税运输的重要枢纽。玩法上可以围绕护送、劫漕、调兵和封锁商路展开，为后续江南地图解锁铺路。

## 5. 地图三：江南水网与西楚故地

### 5.1 地图定位

- **对应区域**：广陵江沿线、旧楚残垣、江南盐路、水寨、士族与遗民网络。
- **视觉主题**：烟雨楼船、断壁宫阙、潮湿青石路、残红灯影。
- **叙事用途**：表现“西楚遗脉未绝、江南豪族各怀心思、文气与杀机并行”的中期扩张感。
- **推荐主路线**：广陵江渡口 → 盐帮黑埠 → 旧楚宫阙 / 亡国旧臣宅 → 西垒壁遗址。

### 5.2 场景节点

1. **广陵江渡口**  
   - 类型：枢纽 / 船运 / 过图节点  
   - 相邻：襄樊水驿、盐帮黑埠、江南绣坊  
   - 这是江南地图的入口，先给玩家“富庶但不太平”的第一印象。玩法上适合承接船票、货运、护送、截杀和消息交换，也是跨图补给点。

2. **盐帮黑埠**  
   - 类型：商店 / 黑市 / 灰色交易  
   - 相邻：广陵江渡口、稻香水寨、亡国旧臣宅  
   - 盐帮势力最适合转化为利润与风险并存的节点，视觉上可做夜船、盐包、码头火把。玩法上能提供高收益走私、稀有物资交易、黑市换牌，但会累积官府追查度。

3. **江南绣坊**  
   - 类型：功能 / 社交 / 礼物资源  
   - 相邻：广陵江渡口、春神湖书肆  
   - 绣坊不是纯商店，而是江南人情网络和礼物经济的起点。玩法上适合产出送礼卡、宴会服饰、名帖等社交资源，降低权谋地图和宗门地图中的交涉难度。

4. **春神湖书肆**  
   - 类型：探索 / 文士 / 舆论  
   - 相邻：江南绣坊、亡国旧臣宅、棠溪剑炉  
   - 书肆天然适合安放文人清谈、密文递送、讽谏文章与秘史残卷。玩法上能解锁舆论牌、策士任务、古籍情报，也可触发“诗文换声望”的柔性成长。

5. **亡国旧臣宅**  
   - 类型：主线 / 势力交涉 / 密谋  
   - 相邻：盐帮黑埠、春神湖书肆、旧楚宫阙  
   - 这里代表西楚遗臣、门客和死士暗流，是江南地图最重要的政治爆点之一。玩法上可出现招揽、策反、旧主遗命与复国筹划，让玩家在人情与现实利益之间做选择。

6. **旧楚宫阙**  
   - 类型：探索 / 主线 / 历史残响  
   - 相邻：亡国旧臣宅、西垒壁遗址  
   - 废宫适合做强叙事场景，用残破壁画、断檐和旧乐声烘托“亡国不亡心”。玩法上可以拾取遗物、触发旧楚人物记忆、解锁隐藏阵营标签或牵出复国资源线。

7. **稻香水寨**  
   - 类型：战斗 / 经营 / 水路控制  
   - 相邻：盐帮黑埠、西垒壁遗址  
   - 水寨节点让江南不只是文戏，也能有地盘争夺与船战味道。玩法上可做帮派围剿、运粮护航、招募水匪转私兵，形成与北凉军旅完全不同的战斗体验。

8. **棠溪剑炉**  
   - 类型：商店 / 锻造 / 江湖支线  
   - 相邻：春神湖书肆、西垒壁遗址  
   - 江南需要一个能承接武器升级与名剑传闻的节点，棠溪剑炉很适合这一职责。玩法上可制作装备、修复名剑、触发剑客订单，也能作为通往武帝城线的前置节点。

9. **西垒壁遗址**  
   - 类型：主线 / 战场遗迹 / 高难挑战  
   - 相邻：旧楚宫阙、稻香水寨、棠溪剑炉  
   - 这是江南地图的终局节点，应当以残战场和执念感压住全图气质。玩法上适合大型连续事件、阵营抉择和强敌挑战，决定玩家是借西楚余绪造势，还是彻底压灭旧火。

## 6. 地图四：宗门群山

### 6.1 地图定位

- **对应区域**：武当、龙虎、徽山、两禅寺等门派与山路节点。
- **视觉主题**：云海栈道、古松石阶、道观香火、佛寺晨钟。
- **叙事用途**：作为中期江湖成长图，负责高手结识、武学传承、宗门立场与江湖名望。
- **推荐主路线**：武当山门 → 小莲花峰 / 太虚宫 → 龙虎山天师府 → 徽山大雪坪 → 两禅寺。

### 6.2 场景节点

1. **武当山门**  
   - 类型：枢纽 / 角色剧情 / 山门试礼  
   - 相邻：小莲花峰、山间驿亭  
   - 武当是最适合承接洪洗象、徐脂虎等情感与命数线索的地方。玩法上可通过礼数、名望和人物关系进入更深层区域，体现“不是谁都能见高人”。

2. **小莲花峰**  
   - 类型：主线 / 奇遇 / 命数  
   - 相邻：武当山门、太虚宫  
   - 小莲花峰适合做带有宿命感的事件，氛围清冷却不空。玩法上可触发天机、心境、角色宿缘牌，带来强增益但往往伴随延迟代价。

3. **太虚宫**  
   - 类型：探索 / 道法 / 支援  
   - 相邻：小莲花峰、山间驿亭  
   - 太虚宫可作为武当内部的功能节点，承接道法、卜算、护身符与心法修习。玩法上适合提供一次性保护效果、气运修正与特殊法术型装备。

4. **龙虎山天师府**  
   - 类型：权谋 / 宗教 / 对抗  
   - 相邻：山间驿亭、伏魔台  
   - 龙虎山不是纯修行地，而是有世俗触角的强势门派。玩法上可做宗教名望、符箓交易、天师法令、对手诘难，让玩家在“借势”与“被收编”之间摇摆。

5. **伏魔台**  
   - 类型：挑战 / 试炼 / 高风险收益  
   - 相邻：龙虎山天师府、断崖观瀑  
   - 伏魔台适合高难战斗与特殊检定，视觉上突出峭壁与雷雨。玩法上可掉落稀有法器、道门标签或声望，但失败会附带伤势、诅咒等长期状态。

6. **徽山大雪坪**  
   - 类型：主线 / 江湖恩怨 / 山庄势力  
   - 相邻：断崖观瀑、山间驿亭、两禅寺山门  
   - 徽山线能承接轩辕家、家族权势与江湖礼法崩坏的戏。玩法上适合宴席试探、比武招婿、家族分裂和夺权事件，兼顾权谋与江湖血气。

7. **两禅寺山门**  
   - 类型：支线 / 修心 / 休整  
   - 相邻：徽山大雪坪、藏经石窟  
   - 两禅寺提供了宗门地图中最稀缺的平静感，是重要的缓冲节点。玩法上可消除负面状态、恢复伤势、触发僧人辩机和因果选择，帮助玩家调整中期压力。

8. **藏经石窟**  
   - 类型：探索 / 经卷 / 隐藏传承  
   - 相邻：两禅寺山门  
   - 石窟适合布置一次性强奖励与解谜型内容，强化“武学不止靠打，也靠悟”的体验。玩法上可获得秘籍、悟性增益、角色专属转职条件或隐藏地图钥匙。

9. **山间驿亭**  
   - 类型：功能 / 驿站 / 游侠交汇  
   - 相邻：武当山门、太虚宫、龙虎山天师府、徽山大雪坪  
   - 宗门地图需要一个便于转场的低压节点，山间驿亭正适合承载行脚僧、道人、镖师、散修的流动。玩法上可刷新游侠事件、接取委托、快速移动，也能遇到限时访客。

10. **断崖观瀑**  
   - 类型：挑战 / 悟道 / 决斗  
   - 相邻：伏魔台、徽山大雪坪  
   - 这是宗门地图中的高名望试剑点，画面感强，适合放强敌与传说人物。玩法上可进行一对一决斗、悟道试炼、名望榜挑战，是通向武帝城地图的重要前置。

## 7. 地图五：东海武帝城

### 7.1 地图定位

- **对应区域**：武帝城及沿海海路、城外礁石、港埠、客栈与黑市。
- **视觉主题**：城墙压海、潮声不绝、海雾迷离、剑意森寒。
- **叙事用途**：作为中后期“武道名望试金石”，也是老黄、李淳罡、剑道角色线的集中爆发区。
- **推荐主路线**：海雾栈桥 → 武帝城城门 → 剑石坪 → 观潮台 → 孤礁试剑处。

### 7.2 场景节点

1. **海雾栈桥**  
   - 类型：入口 / 探索 / 预热  
   - 相邻：临海客栈、武帝城城门、旧船坞  
   - 玩家初到武帝城，不应直接入城，而应先感受到海雾、潮气与压迫感。玩法上可布置传闻收集、船票消耗、初见高手与物资准备。

2. **临海客栈**  
   - 类型：功能 / 情报 / 休整  
   - 相邻：海雾栈桥、东海黑市  
   - 客栈是最适合堆叠“所有人都在等看一场好戏”氛围的地方。玩法上提供休整、传闻、赌局、悬赏与挑战报名，让武帝城名望线具备节奏蓄力。

3. **武帝城城门**  
   - 类型：主线 / 门槛 / 身份检定  
   - 相邻：海雾栈桥、剑石坪、潮神楼  
   - 这不是普通城门，而是江湖地位的筛网。玩法上可通过名望、推荐信、战绩或强行闯关进入，决定后续城内资源和敌意等级。

4. **剑石坪**  
   - 类型：挑战 / 擂台 / 名望  
   - 相邻：武帝城城门、观潮台、潮神楼  
   - 剑石坪适合承接高频短战斗与连胜挑战，是玩家证明自己最直接的方式。玩法上可做擂台连战、下注、名望排名与观众反馈，奖励与风险同步递增。

5. **潮神楼**  
   - 类型：支线 / 名流 / 宴谈  
   - 相邻：武帝城城门、剑石坪、东海黑市  
   - 潮神楼是江湖名流、商贾、世家子集会之地，适合用来平衡纯战斗节奏。玩法上能触发宴谈拉拢、赌约、切磋邀约、名器交易与流言扩散。

6. **旧船坞**  
   - 类型：探索 / 海路 / 隐秘任务  
   - 相邻：海雾栈桥、海商大船  
   - 船坞让武帝城地图从“擂台挑战”扩展到“海上暗线”，内容更丰富。玩法上可进行偷渡、追踪、走私、海盗遭遇和遗物打捞。

7. **东海黑市**  
   - 类型：商店 / 黑市 / 稀有装备  
   - 相邻：临海客栈、潮神楼、旧船坞  
   - 黑市负责武帝城最诱人的资源循环，适合放名剑碎片、禁药、旧符箓和高价情报。玩法上可拍卖稀有物、卖战利品、接危险委托，也可能引来高手盯上。

8. **观潮台**  
   - 类型：主线 / 顶级对话 / 悟剑  
   - 相邻：剑石坪、孤礁试剑处  
   - 观潮台适合承接强叙事与高手对话，是武帝城“静中见杀机”的代表。玩法上可以通过观潮悟剑、对谈问心、角色陪同触发特殊增益，为终极决战做铺垫。

9. **海商大船**  
   - 类型：交易 / 远行 / 外域素材  
   - 相邻：旧船坞  
   - 海商大船可给玩家一种世界更大的感觉，也能引入罕见素材与外来人脉。玩法上适合高价交易、时限交换、海外奇货与特殊支线入口。

10. **孤礁试剑处**  
   - 类型：终局挑战 / 角色线高潮  
   - 相邻：观潮台  
   - 这是武帝城地图最顶级的节点，应当有一种“潮起潮落间只剩一剑”的肃杀感。玩法上适合作为传奇对决、名剑归属、老黄或李淳罡相关高潮事件的落点。

## 8. 地图六：北莽南朝草原

### 8.1 地图定位

- **对应区域**：北莽南朝边境、王帐势力、草原商路、军营和雪原据点。
- **视觉主题**：狼旗、冰湖、草浪、金帐、夜雪与火堆。
- **叙事用途**：承担后期大战前夜、谍战、斥候、会盟、异域强敌压力。
- **推荐主路线**：雪原斥候路 → 南朝边镇 → 蛛网暗市 / 龙腰州牧场 → 狼庭王帐 → 胡骑军营。

### 8.2 场景节点

1. **雪原斥候路**  
   - 类型：入口 / 探索 / 高压移动  
   - 相邻：葫芦口前哨、南朝边镇、冰湖渡口  
   - 这是进入北莽的第一道筛选，先把“气候即敌人”的感受打出来。玩法上强调生存、潜行、补给和斥候检定，失败会直接影响后续数个节点状态。

2. **南朝边镇**  
   - 类型：支线 / 民情 / 敌境试探  
   - 相邻：雪原斥候路、蛛网暗市、龙腰州牧场  
   - 边镇能让玩家看到北莽并不只是军队，也有普通人的生活与不同立场。玩法上可触发收买商人、伪装潜伏、散布谣言、策反部族等复杂选择。

3. **蛛网暗市**  
   - 类型：黑市 / 谍战 / 情报  
   - 相邻：南朝边镇、狼庭王帐、冰湖渡口  
   - “蛛网”非常适合转成情报势力网络，是北莽地图最重要的谍报节点。玩法上可购买敌军部署、交换俘虏、安排内应，也可被反钓鱼导致暴露身份。

4. **龙腰州牧场**  
   - 类型：资源 / 骑军 / 补给  
   - 相邻：南朝边镇、胡骑军营  
   - 北莽强在骑军，牧场节点理应成为战争资源的重要来源。玩法上可破坏马群、偷取战马、收买牧主或争夺补给线，直接影响大战难度。

5. **冰湖渡口**  
   - 类型：挑战 / 移动 / 限时事件  
   - 相邻：雪原斥候路、蛛网暗市、佛窟营地  
   - 冰湖是天然的高风险通道，具有强烈的画面性和时间压力。玩法上可做限时追击、破冰埋伏、救援撤离与宝物打捞，让移动本身变成玩法。

6. **佛窟营地**  
   - 类型：探索 / 异术 / 争议信仰  
   - 相邻：冰湖渡口、狼庭王帐  
   - 北莽不只是铁骑，也有自己的精神秩序与怪异秘闻，这个节点负责补足文化层次。玩法上可出现异术修士、祭祀、密咒、信仰冲突和非常规增益。

7. **狼庭王帐**  
   - 类型：主线 / 会盟 / 顶级权谋  
   - 相邻：蛛网暗市、佛窟营地、胡骑军营  
   - 王帐是北莽地图的政治中心，和太安城遥相呼应，但风格更野更直接。玩法上适合做大人物会面、离间、结盟、斩首计划与真假使者事件，是战争走向的关键分叉点。

8. **胡骑军营**  
   - 类型：主线 / 战斗 / 破阵  
   - 相邻：龙腰州牧场、狼庭王帐、拒北城方向出口  
   - 军营节点必须直接让玩家感到大战迫近，是北莽地图的硬核战斗区。玩法上可做夜袭、偷旗、焚粮、决战前哨与多阶段军阵检定。

9. **祭狼高坡**  
   - 类型：支线 / 仪式 / 士气战  
   - 相邻：狼庭王帐、胡骑军营  
   - 高坡上的祭礼既能增强异域感，也能赋予战争更多心理与精神层面的玩法。玩家可选择破坏祭礼、借势伪装、献祭换取力量，强收益伴随高副作用。

10. **拒北城前线**  
   - 类型：终局 / 会战 / 阵营结算  
   - 相邻：胡骑军营、北凉最终战线  
   - 该节点是北莽与北凉两张战争地图的汇合口，负责承接真正的大规模战役。玩法上建议采用连续多阶段事件、兵力资源结算、重要角色生死抉择和多结局分支。

## 9. 地图七：南疆瘴域

### 9.1 地图定位

- **对应区域**：茶马古道南端、湿热山林、蛊术部落、土司地盘、溶洞暗河。
- **视觉主题**：雨林毒雾、彩羽祭面、藤桥火盆、潮湿岩壁。
- **叙事用途**：作为后期特殊地图，负责提供与中原、北地完全不同的威胁类型与稀有资源。
- **推荐主路线**：茶马古道口 → 瘴雾栈道 → 巫蛊寨 / 药王谷 → 土司王庭 → 雨林神庙。

### 9.2 场景节点

1. **茶马古道口**  
   - 类型：入口 / 交易 / 衔接节点  
   - 相邻：离阳边贸口、瘴雾栈道、边陲驿馆  
   - 这是进入南疆前最后一个相对正常的节点，适合让玩家补给和调整队伍。玩法上承接货物交换、向导雇佣、地图购买与南疆传闻收集。

2. **边陲驿馆**  
   - 类型：功能 / 驿站 / 中转  
   - 相邻：茶马古道口、南疆黑市  
   - 边陲驿馆负责在诡谲氛围前提供一个脆弱的安全点，让南疆冒险更有层次。玩法上可休整、存放资源、招募临时向导，也可能发生夜袭、失踪等突发事件。

3. **瘴雾栈道**  
   - 类型：挑战 / 生存 / 限时推进  
   - 相邻：茶马古道口、巫蛊寨、药王谷  
   - 栈道要突出南疆最直观的危险：空气、地形、视野都在伤害玩家。玩法上可以持续施加中毒、迷路、惊扰毒虫等状态，逼迫玩家重视药材与向导配置。

4. **巫蛊寨**  
   - 类型：主线 / 交涉 / 异术  
   - 相邻：瘴雾栈道、南疆黑市、土司王庭  
   - 巫蛊寨不是单纯敌对势力，更适合设计成“可怕但可合作”的灰度节点。玩法上可以交易蛊术、解咒、下咒、换取控制类卡牌，也可能因失礼而被整寨敌视。

5. **药王谷**  
   - 类型：资源 / 回复 / 稀有材料  
   - 相邻：瘴雾栈道、暗河溶洞  
   - 药王谷承担南疆地图的生存补给职能，也为玩家提供长线对抗毒雾的手段。玩法上可采药、炼制解毒丹、恢复伤势、制作一次性高价值药剂。

6. **南疆黑市**  
   - 类型：商店 / 禁物 / 信息差  
   - 相邻：边陲驿馆、巫蛊寨、白骨祭场  
   - 黑市最适合容纳见不得光的交易，比如蛊虫、尸毒、奇矿和奴隶情报。玩法上可高价买卖稀有物、处理脏物、接暗杀委托，但会提高被追踪与反噬概率。

7. **暗河溶洞**  
   - 类型：探索 / 潜行 / 秘藏  
   - 相邻：药王谷、雨林神庙  
   - 溶洞让南疆地图拥有立体空间，不止是地表丛林。玩法上可做潜水、寻宝、机关、地底追逐与隐藏 boss，是很好的高回报探索节点。

8. **土司王庭**  
   - 类型：主线 / 政治 / 盟约  
   - 相邻：巫蛊寨、白骨祭场、雨林神庙  
   - 王庭承担南疆的权力结构展示，让南疆不是只有“野”而没有“治”。玩法上适合做部族会盟、婚盟、贡赋、叛乱镇压和外交施压，强调区域政治博弈。

9. **白骨祭场**  
   - 类型：挑战 / 仪式 / 高副作用增益  
   - 相邻：南疆黑市、土司王庭  
   - 白骨祭场负责把南疆的危险感推到最极端，是高风险高回报的典型节点。玩法上可通过献祭、破坏祭礼、救人或抢夺圣物获取强力效果，但通常会积累长期诅咒或阵营仇恨。

10. **雨林神庙**  
   - 类型：终局 / 隐藏线 / 异域秘宝  
   - 相邻：暗河溶洞、土司王庭  
   - 这是南疆地图的最终高潮场景，应突出“神秘、古老、不属于中原秩序”的异质感。玩法上适合放隐藏结局线、神器获取、远古真相与特殊角色觉醒事件。

## 10. 地图解锁逻辑建议

### 10.1 推荐节奏

建议采用“**1 张开局主图 + 2 张中期开枝图 + 2 张中后期高压图 + 2 张后期特殊图**”的结构，既保证叙事推进，又给玩家自由选择扩张顺序。

### 10.2 解锁表

| 地图 | 建议阶段 | 解锁条件建议 | 设计理由 |
|------|----------|--------------|----------|
| 北凉道与边塞 | 开局 | 初始开放 | 作为玩家阵营大本营，先教会基础玩法、角色关系和边塞氛围。 |
| 离阳京畿与中原官道 | 第一阶段 | 完成“清凉山王府”主线 2 次以上，且声望达到 20 | 玩家在北凉站稳后，才有资格进入更高层的庙堂博弈。 |
| 宗门群山 | 第一阶段并行 | 触发“芦苇荡渡口”或“听潮亭”中的江湖线事件，且至少结识 1 名江湖角色 | 让玩家从北凉内部自然转去江湖，不显得突兀。 |
| 江南水网与西楚故地 | 第二阶段 | 完成“襄樊水驿”主线，且拥有至少 1 张情报/文士/旧楚相关标签卡 | 江南与西楚更适合在玩家已理解权谋和人脉后开放。 |
| 东海武帝城 | 第二阶段高难 | 宗门群山名望达到阈值，或完成“断崖观瀑”高阶挑战 | 武帝城应该是“你有资格去试剑”的奖励，而不是普通旅游点。 |
| 北莽南朝草原 | 第三阶段 | 完成“流州边关”与“葫芦口前哨”战争线，且全局声望 / 军功达到阈值 | 北莽是战争主线，必须在前线压力充分建立后开放。 |
| 南疆瘴域 | 第三阶段并行 / 隐藏 | 完成江南或离阳地图中的暗线任务，并获得“异术”“密使”或“茶马道”类标签 | 南疆更适合作为特殊线和隐藏线，解锁方式应偏隐秘，不走纯线性。 |

### 10.3 进一步的系统建议

- **地图不建议一次性全开**：最多保持 2~3 张地图同时活跃，保证玩家能理解局势重点。
- **前线地图要与资源地图互锁**：北莽大战前，最好要求玩家在离阳 / 江南 / 北凉中至少完成两条资源线，否则前线会显著更难。
- **隐藏地图依赖标签而非纯数值**：像南疆、武帝城这类特殊地图，建议更看重角色、事件、装备、名望标签组合，增强重玩价值。
- **驿站节点要承担路线压缩**：`北凉暗驿`、`青鸟官驿`、`山间驿亭`、`边陲驿馆` 这类节点建议进入快速旅行系统，减少纯跑图疲劳。

## 11. 对现有场景系统的落地建议

### 11.1 节点类型映射

- **主线剧情 / 角色邂逅 / 权谋事件**：优先使用 `event`
- **校场、擂台、前线、试炼**：优先使用 `challenge`
- **马市、黑市、绣坊、剑炉、客栈商人**：优先使用 `shop`

### 11.2 地图内节点配比建议

每张地图建议采用以下结构：

- 2~3 个 **主线推进节点**
- 2~3 个 **战斗 / 高风险节点**
- 2 个 **功能节点**（商店 / 驿站 / 回复）
- 2~4 个 **支线 / 探索节点**

这样可以保证地图既有方向感，也有足够的卡牌调度空间。

### 11.3 后续拆 JSON 的优先级

建议优先把以下 8 个节点做成可玩样板：

1. 清凉山王府  
2. 芦苇荡渡口  
3. 白马游弩校场  
4. 太安城朱雀门  
5. 尚书台  
6. 武当山门  
7. 武帝城城门  
8. 葫芦口前哨

这 8 个节点能最快搭出“北凉—庙堂—江湖—边战”的完整核心骨架。
