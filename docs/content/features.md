# 功能系统

本文档定义游戏的各项功能系统。

## 核心功能一览

| 功能 | 说明 |
|------|------|
| 声望 | 影响游戏事件的触发 |
| 金骰子 | 检定时消耗一枚可增加一点成功 |
| 处刑日 | 倒计时归0时，若手牌区还持有苏丹卡，游戏结束 |
| 下一天 | 推进游戏时间 |
| 上一天 | 时间回溯，有次数限制 |
| 俺寻思 | 将卡牌置入思考槽，触发该卡牌的思考事件 |
| 卡牌区 | 所有可支配的卡牌都显示在这里 |

---

## 声望系统

### 声望值

- **范围**：0 - 100
- **初始值**：50（中立）

### 声望等级与效果

| 声望区间 | 等级 | key | 效果 |
|----------|------|-----|------|
| 0-19 | 卑微 | `humble` | 部分场景拒绝参与，商店涨价50% |
| 20-39 | 平民 | `common` | 无特殊效果 |
| 40-59 | 受尊敬 | `respected` | 解锁中级场景，NPC好感度+1 |
| 60-79 | 显赫 | `prominent` | 解锁高级场景，商店打折10% |
| 80-100 | 传奇 | `legendary` | 解锁隐藏场景，特殊对话选项 |

### 声望变化

- **增加**：完成正义事件、帮助NPC、捐赠金币
- **减少**：失败事件、背叛NPC、偷窃行为

## 俺寻思系统

将卡牌置入思考槽，触发该卡牌的思考事件。

### 基础规则

- **每日次数**：3次（可配置），黎明阶段重置
- **触发方式**：将一张卡拖入思考槽并确认，消耗1次机会
- **防刷规则**：每张卡每天最多触发1次俺寻思（脚本可配置覆盖）

### 产出类型

由脚本定义，可能包括：
- 叙事对话
- 资源变化（金币/声望/回溯次数/金骰子）
- 卡牌生成/升级/移除
- 标签变化
- 场景解锁

### 存档字段

- `think_charges`：今日剩余俺寻思次数
- `think_used_today`：今日已使用俺寻思的卡牌ID列表

## 成就系统

### 界面布局

- **左侧区域**：成就列表导航
  - 顶部显示成就分类名称（如"一千零一夜"）和点数
  - 列表展示多个成就项，用金色分割线分隔
  - 已完成项显示金色勾选标记
- **右侧区域**：成就详情
  - 成就标题（暗红色横幅）
  - 成就图标（徽章样式，显示奖励数量如"+1"）
  - 成就描述和背景故事
  - 引用台词
  - 完成条件（带勾选标记）
  - 物品插图

### 成就类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 剧情成就 | 与主线故事相关 | 苏丹的戒指 |
| 轮回成就 | 涉及多周目玩法 | 轮回之人 |
| 神话成就 | 与神话元素相关 | 诸神降临 |
| 选择分支成就 | 涉及不同结局 | 你想怎么死 |
| 隐藏成就 | 需要特定条件触发 | 沉入梦中 |
| 文学典故成就 | 与文学典故相关 | 蔷薇花下死 |
| 因果报应成就 | 与游戏行为后果相关 | 宛宛相报 |

### 奖励系统

- 完成成就获得命运点
- 积分系统量化奖励
