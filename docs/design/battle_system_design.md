# 《觅长生》研究与《雪中悍刀行》战斗系统设计方案

## 0. 文档目标

本文档分为两部分：

1. 深入拆解《觅长生》的战斗系统，提炼其真正值得借鉴的机制核心。
2. 结合《雪中悍刀行》的武学体系、人物风格与当前项目已有的卡牌/场景/事件框架，设计一套适合本游戏的战斗系统方案。

本文是**纯设计文档**，不涉及代码实现。

## 2. 第一部分：深入研究《觅长生》战斗系统

## 2.1 一句话总结

《觅长生》的战斗并不是传统抽牌 Deckbuilder，而是：

> **以“灵气随机分布 + 五行出手顺序 + 功法被动联动 + 状态叠层”构成的回合制资源编排战斗。**

它的魅力不在“单张牌强”，而在“**顺序、资源转化、被动触发、状态联动、流派闭环**”。

---

## 2.2 核心战斗机制

### 2.2.1 回合制如何运作

根据公开攻略与 WIKI 信息，《觅长生》的标准战斗可概括为：

1. **战前探查**
   - 可用神识探查对手装备、功法、主动技能与被动技能。
   - 这意味着战斗并非完全盲打，存在明显的“知己知彼”层。

2. **战斗开始**
   - 开局被动生效：
     - 部分功法给开局灵气
     - 部分功法给开局状态
     - 部分装备直接赋予护罩、驭X、毒、灼烧等

3. **先手判定**
   - 遁速更快者先出手。
   - 遁速还决定能否成功逃跑。

4. **每回合资源刷新**
   - 每回合会获得固定总量灵气。
   - 灵气按五行权重随机分配。
   - 权重受灵根与构筑影响。

5. **玩家行动**
   - 玩家不是“从牌库抽到技能”，而是从自己装配好的主动神通/武器中选择出手顺序。
   - 技能消耗对应属性灵气。
   - 顺序会影响五行相生相克与被动触发。

6. **回合结算**
   - 计算伤害、状态增减、回合开始/结束触发。
   - 护罩、灼烧、中毒、剑气、化焰等状态在不同时间点结算。

### 2.2.2 出牌/技能选择机制的本质

《觅长生》的“出牌感”来自三层：

- **资源层**：当前回合抽到什么属性灵气
- **顺序层**：先用哪一个属性技能，再接哪一个属性技能
- **联动层**：这次出手会不会触发功法、悟道、武器、状态的额外收益

因此它不是“单回合战术题”，而更像：

> **在有限资源与随机属性下，把自己的流派引擎跑起来。**

---

## 2.3 资源系统

## 2.3.1 灵气是核心资源

公开资料能明确验证：

- 战斗的核心资源是**灵气**
- 灵气分为五种属性：**金、木、水、火、土**
- 每回合获得的灵气总量固定，但属性构成随机
- 灵气可以保留，但保留上限有限

常见显示方式如 `15/4`，可理解为：

- 当前拥有 15 点灵气
- 回合结束最多保留 4 点

### 2.3.2 灵气的几个关键设计点

1. **资源随机，但不是纯随机**
   - 灵根、功法、装备、属性权重会影响更容易获得哪类灵气。

2. **可以屯，但不能无限屯**
   - 灵气能留存，带来“蓄爆发”的策略空间。
   - 但保留上限限制了无脑龟缩。

3. **存在资源转化手段**
   - 属性转换类神通可以把无用灵气转化为所需灵气。
   - 这让“坏手牌”并不完全死。

4. **资源回流是高阶构筑核心**
   - 高级流派的本质不是“伤害变高”，而是“出手越多，回得越多”，形成永动或半永动闭环。

### 2.3.3 法力/生命/神识等次级资源

从公开资料看：

- **生命**：承伤资源
- **神识**：影响神识类技能、探查与特定战斗表现
- **遁速**：先手与撤退资源
- **护罩/蓄势/剑气/魂等**：本质上是战斗内暂存资源或状态资源

这说明《觅长生》不是单资源系统，而是：

> **“灵气主资源 + 指示物次级资源 + 状态资源”三层结构。**

## 2.4 五行系统

## 2.4.1 五行不是背景设定，而是战斗骨架

五行在《觅长生》中承担了四个职责：

1. **资源类型**
2. **技能属性**
3. **连招判定依据**
4. **流派分类基础**

### 2.4.2 相生相克的实际作用

公开攻略信息显示：

- 相邻五行更容易形成“相生”
- 不相邻五行更容易形成“相克”
- 正确的出手顺序可提升伤害或触发额外段数/效果

这里最重要的不是传统文化解释，而是战斗上的两个结果：

1. **同样一套技能，不同顺序会打出不同收益**
2. **五行关系让“顺序”本身成为一种资源**

### 2.4.3 五行的优秀之处

- 让构筑有明确风格标签
- 让随机资源有可规划方向
- 让连招不依赖复杂输入，而依赖“前后手逻辑”
- 既能做单属性专精，也能做双属性桥接

### 2.4.4 五行的缺点

- 新手理解成本高
- 强依赖具体流派功法说明
- 当资源不顺时容易“想打的招打不出来”

## 2.5 功法/技能系统

## 2.5.1 主动与被动的分工

从资料可归纳出：

- **神通/武器**：偏主动
- **功法**：偏被动
- **悟道**：偏流派强化与规则重写

这三者叠加，形成一套完整战斗身份。

### 2.5.2 功法的效果模式

WIKI 中可明确观察到，功法效果主要分为几类：

1. **开局资源型**
   - 开局获得灵气
   - 开局给护罩/缠绕/中毒/神殇等

2. **回合触发型**
   - 回合开始获得资源/状态
   - 回合结束回血/掉血/叠层

3. **出手触发型**
   - 每次使用技能有概率回灵气
   - 每次造成某属性伤害，追加一次额外技能
   - 每次伤害额外叠易伤

4. **顺序判定型**
   - 例如水后接木触发额外负面状态
   - 或某一属性技能接另一属性技能形成连击

5. **转化增伤型**
   - 溢出治疗转伤害
   - 把受伤、回血、蓄势等转为输出

6. **启动引擎型**
   - 不是直接伤害高，而是帮助整个流派“转起来”

### 2.5.3 神通/技能的效果类型

主动技能常见效果包括：

- 直接伤害
- 多段伤害
- 属性转化
- 附加状态
- 获得指示物
- 消耗已有指示物换爆发
- 自身防御
- 拉高下次伤害

### 2.5.4 连招 / Combo 的核心形式

《觅长生》的 Combo 不是动作输入，而是四类系统连携：

1. **属性连携**
   - 如水接木、金接土等

2. **状态连携**
   - 先叠霜冻，再吃裂冰增伤
   - 先叠易伤，再靠多段频繁结算

3. **资源连携**
   - 先转气，再爆发
   - 先回气，再继续出手

4. **功法连携**
   - 一个功法给出手次数
   - 一个功法给回气
   - 一个功法给每段增伤
   - 三者闭环后流派才成立

## 2.6 状态 / Buff 系统

## 2.6.1 《觅长生》状态设计的最大特点

其状态系统不是“加攻减防”那么简单，而是：

> **大量以“每层 = 1 个明确的数值效果”表达，且与出手次数、结算时机强绑定。**

这让状态天然适合做构筑核心。

### 2.6.2 可明确验证的典型状态

根据公开装备/炼器词条资料，可整理为：

| 状态 | 已验证效果 |
|------|------------|
| 护罩 | 每层抵挡 1 点伤害，不主动移除 |
| 易伤 | 本回合受到的技能伤害增加 |
| 流血 | 回合结束按已损生命比例受伤，触发后减层 |
| 疗 | 回合结束回血，触发后减层 |
| 霜冻 | 技能基础伤害下降，每次使用技能移除一层 |
| 缠绕 | 本回合每用一次技能，自身受 1 点伤害 |
| 中毒 | 回合结束受伤，触发后减层 |
| 灼烧 | 回合开始受伤，触发后减层 |
| 化焰 | 受技能伤害时反弹火伤，且回合结束追加灼烧；用非火技能会清空 |
| 减伤 | 本回合受到的技能伤害减少 |
| 剑气 | 回合结束造成剑系伤害，不主动移除 |
| 反伤 | 本回合受技能伤害时反弹伤害 |
| 神殇 / 凝神 | 改变神识数值，不主动移除 |
| 速 / 缚足 | 改变遁速，不主动移除 |

### 2.6.3 状态设计的优秀原则

《觅长生》的状态之所以有深度，是因为它同时满足：

- **易理解**：每层效果很直观
- **可堆叠**：可以围绕层数构筑
- **有时机差**：回合开始、回合结束、受击时、出手时各不相同
- **能反制**：有辟毒、辟火、辟水、移除易伤等手段

### 2.6.4 叠层机制的战略意义

叠层并不只是“数值膨胀”，它承担三类博弈：

1. **爆发预热**
2. **持续压制**
3. **逼迫对手改变出手顺序**

这也是《觅长生》很多高深流派的本质来源。

## 2.7 构筑体系

## 2.7.1 玩家如何构建自己的流派

从流派攻略中可以明显看出，《觅长生》的构筑不是“带几张强牌”，而是五个层面共同决定：

1. **灵根/权重**
   - 决定更容易抽到什么灵气

2. **功法组合**
   - 决定引擎和被动机制

3. **主动技能组合**
   - 决定出手方式和爆发形态

4. **悟道**
   - 决定规则强化方向

5. **装备**
   - 决定补短板、回气、清状态、加权重、提高启动稳定性

### 2.7.2 构筑类型

基于资料可归纳出几条主要构筑轴：

- **一击流**
  - 核心是蓄势、增伤、单次爆发

- **多段流**
  - 核心是高频触发、每段附加收益、易伤/负面状态放大

- **持续伤害流**
  - 核心是毒、灼烧、流血等

- **防守反击流**
  - 核心是护罩、减伤、反伤、蓄势

- **资源永动流**
  - 核心是“出手越多回得越多”

### 2.7.3 《觅长生》构筑最值得借鉴的地方

不是流派数量多，而是：

> **每个成熟流派都有“启动条件—运转过程—终结方式—被谁克制”的完整生态。**

## 2.8 战斗节奏

## 2.8.1 节奏特征

玩家评价中反复出现两个判断：

1. **策略空间很大**
2. **战斗容易偏慢**

### 2.8.2 为什么会慢

主要有四个原因：

1. 资源属性随机，常常要先“整理手气”
2. 很多流派需要预热
3. 防御流和高减伤敌人会拉长战斗
4. 多段流操作次数多，回合内动作密集

### 2.8.3 关键转折点

《觅长生》的关键转折通常出现在：

- 成功把资源转成目标属性时
- 某个核心状态叠到阈值时
- 回气引擎启动时
- 对手护罩/减伤被击穿时
- 自己成功从防守切入爆发时

### 2.8.4 爆发时机

常见爆发窗口：

- 单次蓄势斩杀
- 多段引擎启动后的连发
- 通过状态（易伤、霜冻、毒、缠绕）放大后集中输出
- 依靠武器/主动法宝在关键回合变招

---

## 2.9 敌人设计

## 2.9.1 普通敌人行为模式

公开资料能观察到两点：

1. 游戏会刻意避免玩家靠龟缩轻松磨死怪
2. 不少敌人带有“越打越强”的成长机制

这类设计能逼玩家主动找击杀窗口。

### 2.9.2 Boss/高难敌人的核心思路

从玩家流派讨论里可以反推敌人设计逻辑：

- 有的 Boss 克制多段流
- 有的 Boss 克制叠层流
- 有的 Boss 用高减伤/特殊反制打断永动
- 有的 Boss 用魔气入体之类规则限制单回合出手次数

这说明《觅长生》的敌人并不只是“面板高”，而是：

> **直接用机制去挑战玩家构筑。**

### 2.9.3 这一点为什么重要

因为当玩家构筑深度提高后，如果敌人只会“加血加攻”，战斗会立刻塌成数值题。

《觅长生》更聪明的地方在于：

- 让敌人变成“特定流派的天敌”
- 倒逼玩家准备第二套解法

## 2.10 数值框架

## 2.10.1 已验证的数值表达风格

尽管本次未找到完整官方总公式，但能确认《觅长生》采用了很强的“**离散层数数值化**”设计：

- 护罩：每层挡 1
- 减伤：每层减 1
- 霜冻：每层减基础伤害
- 缠绕：每次出手受 1
- 剑气：回合结束定额伤害
- 易伤：增加技能伤害

也就是说，它大量使用：

> **“基础值 + 层数修正 + 时机结算”**

而不是完全依赖复杂乘区。

### 2.10.2 可归纳出的伤害结算层

结合状态定义与流派说明，可以把《觅长生》的伤害逻辑归纳为：

1. **技能基础伤害**
2. **属性/灵气/连击加成**
3. **功法/悟道/装备增伤**
4. **目标易伤/霜冻/特殊状态修正**
5. **目标减伤/护罩/防御型状态结算**
6. **多段逐段独立结算**

其中“多段逐段结算”极其关键，因为：

- 每一段都可能触发功法
- 每一段都可能叠状态
- 每一段都可能被减伤克制

这直接决定了多段流与一击流是两种完全不同的数值生态。

### 2.10.3 防御机制

从资料看，防御不是单一护甲，而是多层：

- 护罩
- 减伤
- 反伤
- 清 debuff
- 降低基础伤害（如霜冻类影响）
- 特定悟道/机制对多段或单段的克制

### 2.10.4 暴击 / 闪避 / 命中

本次未查到公开且稳定的完整暴击/闪避总公式，因此不强行写死。

但从系统风格看，《觅长生》的重点并不在传统 RPG 那种高权重暴击，而在：

- 先手
- 出手顺序
- 状态阈值
- 资源闭环
- 特定构筑反制

也就是说，它更偏“机制制胜”，而不是“赌脸制胜”。

---

## 2.11 对《觅长生》的抽象结论

如果把《觅长生》战斗系统抽象成设计语言，可以总结为六句话：

1. **随机资源必须能被构筑驯服。**
2. **出手顺序本身就是战术资源。**
3. **被动联动要比单张强牌更重要。**
4. **状态要能成为流派骨架，而不是装饰。**
5. **敌人必须按“克制构筑”来设计。**
6. **高深度战斗的风险是节奏拖慢，必须主动做节奏治理。**

## 3. 第二部分：结合《雪中悍刀行》设计战斗系统

## 3.1 设计总纲

我们的目标不是做“修仙五行皮肤替换版”，而是做：

> **一套有《觅长生》构筑深度，但气质上属于《雪中悍刀行》的武侠战斗系统。**

这套系统需要同时满足：

- 有武侠味，而不是仙侠味
- 有人物辨识度，而不是纯职业模板
- 可与现有卡牌/场景/事件系统无缝衔接
- 战斗深度足够，但节奏比《觅长生》更利落

---

## 3.2 我们的核心设计原则

### 原则 1：从“五行”改为“武意”

《觅长生》的五行强在“顺序关系”，不一定非得保留元素语义。

我们将其替换为更武侠的六类**武意**：

- **刚**：硬打、破势、正面压制
- **柔**：卸力、借力、化劲、回转
- **疾**：快、轻、追击、先手
- **重**：蓄势、破甲、镇压、硬直
- **奇**：诈、变招、暗器、突袭
- **玄**：心法、术法、阵机、天象

这六类武意既能映射兵器门类，也能映射人物心法。

### 原则 2：用“招路相承/相制”替代五行相生相克

我们建立一个六意轮：

**刚 → 柔 → 重 → 奇 → 玄 → 疾 → 刚**

规则如下：

- **相承**：接在上一路之后，获得顺势收益
- **相制**：打克制对方上一招的路数，获得破绽收益

具体对应：

| 关系 | 结果 |
|------|------|
| 相承 | +1 武意、+1 连击、+额外效果触发率 |
| 相制 | 施加【破绽】、削减【架势】、额外伤害 |

这样既保留《觅长生》的“顺序深度”，又更贴合武侠“招路、意气、借势、破招”语境。

### 原则 3：战斗应该更短、更狠、更像名场面

相比《觅长生》，我们的目标节奏是：

- 普通战：**3~5 回合**
- 精英战：**5~7 回合**
- Boss 战：**7~10 回合**

核心转折点更早出现，避免无效磨血。

### 原则 4：角色比流派更重要

《雪中悍刀行》是强人物 IP。

所以我们的构筑不能只围绕“门派流派”，还必须围绕：

- 人物天赋
- 代表招式
- 人物关系
- 名场面触发

## 3.3 武学体系映射

## 3.3.1 六大武学门类与机制映射

| 武学体系 | 人物代表 | 对应主武意 | 机制定位 | 系统特征 |
|----------|----------|------------|----------|----------|
| 剑道 | 李淳罡、邓太阿、老黄 | 疾 / 玄 | 连击、破招、剑意终结 | 先手、高连段、破势、斩护体 |
| 刀法 | 徐凤年、齐练华 | 刚 / 重 | 蓄势、斩杀、血线压制 | 单段高爆发、收割、越战越凶 |
| 枪法 | 徐龙象、徐偃兵系风格 | 刚 / 疾 | 突进、贯穿、阵线压制 | 列伤、破阵、前排压制 |
| 拳法 / 体术 | 王仙芝、徐龙象体魄侧 | 刚 / 重 / 柔 | 反制、霸体、贴身压制 | 韧性高、拆多段、近身强控 |
| 术法 / 阵法 | 洪洗象、黄龙士 | 玄 / 柔 | 预判、延时、地形改写 | 调整资源、封招、领域压制 |
| 暗器 / 轻功 | 温华偏江湖、红薯偏策应 | 奇 / 疾 | 先手、偷势、打断、收尾 | 机动、削弱、点杀后排 |

### 3.3.2 每个体系的战斗身份

#### 剑道

- 强在“见缝插针”
- 不一定面板最厚，但最擅长**破招、抢节奏、切爆发**
- 核心资源：**剑意**
- 核心状态：**破绽、剑鸣、飞剑、斩势**

#### 刀法

- 强在“势大力沉”
- 不靠多段，而靠“前面蓄、后面断”
- 核心资源：**刀势**
- 核心状态：**压迫、断脉、血勇**

#### 枪法

- 强在“直线突破”和“阵线控制”
- 擅长打前排、打列伤、打护卫体系
- 核心资源：**枪劲**
- 核心状态：**贯阵、挑飞、震步**

#### 拳法 / 体术

- 强在“以身压人”
- 既能坦，又能拆对手多段和轻型流派
- 核心资源：**罡劲**
- 核心状态：**霸体、震伤、反震、金刚**

#### 术法 / 阵法

- 强在“改变战斗规则”
- 不是直接最高伤，而是改变资源、回合、站位、时机
- 核心资源：**道息**
- 核心状态：**阵眼、天象、封脉、借风借雷**

#### 暗器 / 轻功

- 强在“节奏切割”
- 可以偷先手、打断、封轻招、杀脆皮
- 核心资源：**机变**
- 核心状态：**标记、残影、失衡、追魂**

## 3.4 资源系统设计

## 3.4.1 总体结构

我们采用三层资源：

1. **武意资源**：六类主资源，替代五行
2. **内息上限**：决定回合结束能保留多少武意
3. **角色专属资源**：剑意、刀势、罡劲、阵眼、残影等

### 3.4.2 六类武意

| 武意 | 武侠含义 | 适合的招式类型 |
|------|----------|----------------|
| 刚 | 正面硬撼、破架 | 刀、枪、拳重击 |
| 柔 | 卸劲借势、牵引 | 化劲、反制、回转 |
| 疾 | 迅捷、逼抢、追击 | 快剑、身法、突刺 |
| 重 | 蓄压、镇杀、破甲 | 蓄刀、重拳、坠击 |
| 奇 | 变招、诡道、暗袭 | 暗器、欺身、偷势 |
| 玄 | 心法、术理、阵机 | 道术、天象、阵法 |

### 3.4.3 获取方式

每回合角色队伍会获得固定总量武意，分布受四个因素影响：

1. **角色武学倾向**
2. **装备武器类型**
3. **已装配心法**
4. **场景地形/天气**

例：

- 李淳罡更容易获得疾、玄
- 徐龙象更容易获得刚、重
- 雪地地图提高重、柔
- 高台/夜战提高奇、疾

### 3.4.4 为什么这比五行更适合武侠

因为武侠核心不是元素，而是：

- 出招风格
- 劲路变化
- 身法快慢
- 破招与借势

“武意”保留了《觅长生》的资源编排深度，同时让玩家天然联想到：

- 这招为什么要接那招
- 这个人物为什么更容易打出这种节奏

### 3.4.5 内息上限

每个角色都有**内息**属性，决定回合结束后可保留的武意总量。

内息来源：

- 体魄
- 智慧
- 心法
- 装备

设计意图：

- 保留蓄爆发乐趣
- 防止无限囤资源
- 让内功型角色和莽夫型角色有明显差别

### 3.4.6 专属资源

为提升人物辨识度，每个门类有专属战斗内资源：

| 体系 | 专属资源 | 用途 |
|------|----------|------|
| 剑道 | 剑意 | 提升多段、破招、绝学阈值 |
| 刀法 | 刀势 | 提升斩杀、断招、压血线 |
| 枪法 | 枪劲 | 提升突刺、贯穿、列伤 |
| 拳法 | 罡劲 | 提升霸体、反震、护体 |
| 术法 | 阵眼 | 激活阵法与延时术式 |
| 暗器/轻功 | 机变 | 触发追击、闪位、偷袭 |

## 3.5 技能 / 招式系统

## 3.5.1 技能分类

建议采用“少而精”的装配结构：

- **6 个主动招式槽**
- **2 个身法/反制槽**
- **3 个心法槽**
- **1 个兵器技槽**
- **1 个绝学槽**

这相当于把《觅长生》的“神通+功法+武器”压缩成更适合当前项目节奏的版本。

### 3.5.2 招式分类定义

| 类别 | 作用 |
|------|------|
| 起手式 | 起状态、抢先手、铺资源 |
| 连段式 | 低耗、多段、吃相承收益 |
| 架势式 | 防御、蓄势、反制 |
| 杀招 | 高耗、高收益、需要窗口 |
| 身法 | 位移、追击、脱离、偷势 |
| 兵器技 | 武器自带主动，低频强效果 |
| 绝学 | 角色/门类招牌技，条件满足后释放 |

### 3.5.3 获取方式

招式来源建议与世界观绑定：

- 角色自带招牌招式
- 场景事件传承
- 师承/秘籍/残卷
- 武库/宝匣
- 羁绊剧情
- 特殊 Boss 战后领悟

### 3.5.4 升级路线

招式成长分三层：

1. **熟练**：数值微增、消耗优化
2. **精通**：附加状态、增加相承收益
3. **宗师**：改变规则或解锁分支形态

例：

- 《两袖青蛇》
  - 熟练：伤害提升
  - 精通：若前一招为疾，追加一段
  - 宗师：若目标处于破绽，直接斩掉所有护体

## 3.6 状态系统设计

## 3.6.1 状态设计原则

借鉴《觅长生》，我们的状态也采用：

- 每层意义明确
- 结算时机固定
- 可叠层
- 可反制
- 能成为构筑核心

但状态数量要控制，避免信息爆炸。

### 3.6.2 核心正面状态

| 状态 | 效果定位 |
|------|----------|
| 护体 | 每层抵挡 1 点技能伤害 |
| 架势 | 提高下一次防御/反制成功率 |
| 剑意 | 提升剑系连段与绝学阈值 |
| 刀势 | 提升刀系斩杀与压血线效果 |
| 罡劲 | 提升霸体、减伤、反震 |
| 阵眼 | 维持阵法效果，消耗后发动术式 |
| 残影 | 提升闪避/追击/偷袭成功率 |
| 血勇 | 低血时增伤增抗 |

### 3.6.3 核心负面状态

| 状态 | 效果定位 |
|------|----------|
| 破绽 | 受到下一次杀招伤害增加 |
| 断脉 | 内息保留下降，回气变差 |
| 失衡 | 无法触发部分相承/反制 |
| 钉伤 | 使用身法时受伤 |
| 寒滞 | 疾类武意收益下降 |
| 灼脉 | 回合开始受伤，并放大重型杀招 |
| 缠足 | 先手与位移能力下降 |
| 封窍 | 无法增长专属资源 |

### 3.6.4 与《觅长生》的对应关系

| 《觅长生》思路 | 我们的转译 |
|----------------|------------|
| 护罩 | 护体 |
| 易伤 | 破绽 |
| 霜冻 | 寒滞 |
| 缠绕 | 钉伤 / 缠足 |
| 中毒 / 灼烧 | 灼脉 / 暗伤 |
| 剑气 | 剑意外放 |
| 减伤 | 罡劲 / 架势 |

### 3.6.5 结算时机

统一分四类：

1. **回合开始**
2. **出招前后**
3. **受击时**
4. **回合结束**

这样 UI 和玩家理解会非常稳定。

## 3.7 战斗流程设计

## 3.7.1 标准战斗流程

### 阶段 A：战前

- 显示敌方身份、武学门类、地形、天气、Boss 特性
- 若有情报卡或事件前置，可额外窥见敌方绝学或弱点

### 阶段 B：开场

- 角色心法、装备、被动先结算
- 根据双方遁速/轻功/奇袭判定先手
- 地形效果入场

### 阶段 C：回合开始

- 吸收本回合武意
- 触发回合开始状态
- 结算持续性伤害/回复

### 阶段 D：行动

玩家本回合可：

- 使用 1~3 次普通招式
- 或 1 次高耗杀招
- 或 1 次绝学
- 可穿插 1 次兵器技/身法技

出招顺序会触发：

- 相承
- 相制
- 专属资源增长
- 角色天赋

### 阶段 E：回合结束

- 结算回合结束型状态
- 结算残留剑意/阵法/反击
- 保留不超过内息上限的武意

### 阶段 F：胜负检查

- 气血归零
- 触发斩杀
- 特殊 Boss 进入下一阶段

## 3.7.2 回合内操作上限

为避免《觅长生》后期多段流过于冗长的问题，我们明确限制：

- 单回合基础出招上限为 3
- 追击、反击、飞剑属于“额外段数”，但总额外触发也设软上限

这样既保留爽感，又不会拖。

## 3.8 构筑体系设计

## 3.8.1 玩家如何构筑自己的战斗风格

构筑由六个层面决定：

1. **主角色**
2. **副角色/援护**
3. **主武学门类**
4. **心法组合**
5. **兵器与奇物**
6. **场景前置准备**

### 3.8.2 三条主要构筑轴

#### 轴一：门类轴

- 纯剑
- 剑刀双修
- 枪拳前排
- 术法控场
- 轻功暗袭

#### 轴二：节奏轴

- 抢先速攻
- 中速连携
- 蓄势爆发
- 防守反制

#### 轴三：角色轴

- 徐凤年统筹队
- 李淳罡剑压队
- 徐龙象冲阵队
- 洪洗象阵控队
- 红薯奇策队
- 老黄守拙反打队

### 3.8.3 典型 build 示例

#### Build A：剑开天门

- 核心角色：李淳罡
- 核心思路：疾/玄相承，快速叠剑意，寻找一击开门
- 优点：破护体、秒脆皮、压 Boss 第一阶段
- 缺点：怕高罡劲、怕封窍、怕拖长线

#### Build B：北凉刀阵

- 核心角色：徐凤年
- 核心思路：刀势蓄压，队友为其制造破绽，斩杀血线
- 优点：单点爆发极高，斩杀爽感强
- 缺点：前两回合较弱

#### Build C：龙虎山河阵

- 核心角色：洪洗象
- 核心思路：阵眼铺场、天象改武意分布、延迟雷法收尾
- 优点：控场强、应对复杂 Boss 稳定
- 缺点：启动慢，怕抢节奏

#### Build D：龙象压城

- 核心角色：徐龙象
- 核心思路：刚/重压制，硬吃对方出手再反压
- 优点：克制多段流、容错高
- 缺点：怕风筝、怕高奇袭

#### Build E：绣口藏针

- 核心角色：红薯
- 核心思路：奇/疾偷势，叠破绽与钉伤，给主 C 开窗口
- 优点：功能性强，配队价值高
- 缺点：单兵终结弱

## 3.9 角色定位设计

## 3.9.1 徐凤年

### 定位

- **全能型主角**
- **借势核心**
- **队伍枢纽**

### 核心机制：借势成刀

徐凤年的特色不应是“最纯粹的刀客”，而应是：

> **最擅长把队友制造的窗口转化为致胜一刀的人。**

机制建议：

- 队友每触发一次相承/相制，徐凤年获得 1 层【借势】
- 消耗借势可把任意武意转为刀势
- 对存在【破绽】的目标，刀系杀招伤害显著提高
- 绝学可根据已消耗的借势层数改变形态

### 配合

- 与红薯配合最好：红薯负责造破绽
- 与洪洗象配合稳定：洪洗象负责做局
- 与老黄配合有情绪价值：老黄负责保驾、喂剑路

### 被克制

- 被高护体高罡劲克制
- 被极快轻功流打断节奏

---

## 3.9.2 李淳罡

### 定位

- **剑道主 C**
- **破势终结者**
- **高爆发高上限**

### 核心机制：剑意开门

李淳罡的战斗感必须体现：

- 天下剑道魁首
- 一招定乾坤
- 破人也破势

机制建议：

- 每次使用疾/玄剑招获得【剑意】
- 若连续两回合未被打断，剑意进入【通天】状态
- 消耗高额剑意释放绝学：
  - 《两袖青蛇》：多段破招
  - 《剑开天门》：斩掉护体并重创

### 配合

- 适合搭配能提供先手和控场的角色
- 与洪洗象形成“天象+剑开天门”大招组合

### 被克制

- 怕罡劲型近身压制
- 怕断脉/封窍导致剑意增长变慢

---

## 3.9.3 徐龙象

### 定位

- **前排坦克**
- **冲阵破军**
- **多段流克星**

### 核心机制：金刚龙象

徐龙象应呈现“傻但强、重但快不起、扛得住又压得死”的战斗质感。

机制建议：

- 受到多段伤害时，每回合前几段自动减伤
- 受击积累【龙象怒】
- 龙象怒可转化为刚/重武意
- 绝学《龙象踏阵》对前排与列阵目标效果极强

### 配合

- 适合保护洪洗象这类慢启动角色
- 适合给徐凤年争取蓄刀时间

### 被克制

- 怕玄系控场和风筝
- 怕持续断脉导致内息不足

---

## 3.9.4 洪洗象

### 定位

- **控场辅助**
- **规则改写者**
- **后手大核**

### 核心机制：天象阵机

洪洗象不应只是“法师伤害位”，而应体现：

- 看似散淡，实则掌控全局
- 借天象、借风雷、借时机

机制建议：

- 开局生成【阵眼】
- 可改写下一回合武意分布
- 可封敌方某一路武意收益
- 可布“延时术式”，在下一回合或敌人行动后触发
- 绝学《请天雷》在目标已有破绽或失衡时伤害翻倍

### 配合

- 与所有慢热主 C 都兼容
- 是队伍战术中枢

### 被克制

- 怕高奇袭直接切后
- 怕极快战斗不给铺场空间

---

## 3.9.5 红薯

### 定位

- **奇策副 C**
- **节奏扰乱者**
- **破绽制造机**

### 核心机制：绵里藏针

红薯在小说中的核心不是正面搏杀，而是：

- 察言观色
- 温柔外表下的锋利
- 最懂局势与人心

机制建议：

- 出手更容易触发奇/疾相承
- 可偷取敌方 1 点武意
- 可施加【失衡】【破绽】【钉伤】
- 若目标已带 2 个以上负面状态，可触发追击

### 配合

- 与徐凤年是天然搭档
- 与李淳罡配合可快速做破绽斩杀

### 被克制

- 怕大范围震荡
- 怕被前排锁死或先手秒杀

---

## 3.9.6 老黄

### 定位

- **守拙剑辅**
- **保护型副核**
- **残局翻盘手**

### 核心机制：藏锋九剑

老黄的战斗应该体现：

- 平时不起眼
- 真到关键时刻比谁都能扛事

机制建议：

- 开局自带【藏锋】
- 每回合第一次为友方承伤时，获得 1 层【剑匣】
- 剑匣可用于：
  - 保护主 C
  - 触发反击
  - 最终释放《剑九》

### 配合

- 与徐凤年、李淳罡都有强剧情联动
- 适合作为“保一手”的副核

### 被克制

- 怕拖到资源耗尽
- 怕持续封专属资源

## 3.10.1 配合关系

| 组合 | 协同点 |
|------|--------|
| 徐凤年 + 红薯 | 红薯造破绽，徐凤年收头 |

| 李淳罡 + 洪洗象 | 天象压制后爆发开门 |
| 徐龙象 + 洪洗象 | 徐龙象抗压保护术法体系 |
| 老黄 + 李淳罡 | 老黄保节奏，李淳罡蓄剑意 |
| 老黄 + 徐凤年 | 强叙事羁绊，保核与反击兼具 |

## 3.10.2 克制关系

| 角色 / 体系 | 克制对象 | 原因 |
|-------------|----------|------|
| 徐龙象 / 拳体 | 多段剑流、暗器流 | 高罡劲、拆多段 |
| 李淳罡 / 剑道 | 护体流、阵法脆皮 | 破势能力强 |
| 红薯 / 奇袭 | 慢启动术法 | 偷节奏、打断铺场 |
| 洪洗象 / 阵法 | 单点爆发刀流 | 通过封招与拖回合限制其窗口 |
| 徐凤年 / 借势刀 | 已被破绽标记的目标 | 收割效率极高 |

## 3.11 敌人设计

## 3.11.1 普通敌人

普通敌人不应拥有复杂规则，只需要 1 个武学标签 + 1 个鲜明特性。

建议模板：

### 江湖散人

- 节奏快，招式少
- 适合做教学敌人

### 北莽骑军

- 枪系、列阵、冲锋
- 适合体现“破阵”和前排压力

### 离阳死士

- 暗器/轻功
- 先手高、血薄、烦人

### 练气士

- 玄系
- 会封脉、改资源、拖节奏

## 3.11.2 精英敌人

每个精英有 1 套明确反制逻辑，例如：

- **铁浮屠校尉**：高护体，克制轻多段
- **蛛网刺客**：极快奇袭，克制慢热术法
- **钦天监术士**：封玄/断脉，克制术法镜像对局

## 3.11.3 Boss 设计原则

Boss 必须是“名场面机制”，不是大血包。

建议 Boss 设计三层：

1. **招牌机制**
2. **阶段转换**
3. **破解方式**

例：

### 王仙芝型 Boss

- 核心机制：拳体霸压、贴身压制、多段抗性
- 阶段 1：金刚不坏，护体极厚
- 阶段 2：开始主动反震和拆招
- 破解思路：不是堆多段，而是先破其架势，再打重型终结

### 韩貂寺型 Boss

- 核心机制：傀儡/丝线/远程操控
- 通过标记与封位来切割队伍
- 破解思路：高机动、打断、斩阵眼

### 龙虎山天师型 Boss

- 核心机制：场地阵法、武意改写
- 破解思路：抢先破阵，或带强反制人物

## 3.12 数值框架设计

## 3.12.1 我们的数值哲学

借鉴《觅长生》，但简化为：

> **基础伤害要清楚，状态层数要直观，乘区数量要少，单回合爆发窗口要明显。**

### 3.12.2 基础战斗数值

建议每个角色在战斗中有以下二级战斗属性：

| 二级属性 | 来源 |
|----------|------|
| 气血 | 体魄 + 装备 + 心法 |
| 内息 | 体魄 + 智慧 + 心法 |
| 先手 | 隐匿 + 生存 + 轻功标签 |
| 命中 | 战斗 + 智慧 |
| 招架 | 战斗 + 体魄 |
| 洞察 | 智慧 + 社交 |
| 术理 | magic 现有字段 + 智慧 |

### 3.12.3 现有八维属性如何映射到战斗

为兼容当前数据结构，建议保留现有 8 维作为底层来源：

| 现有属性 | 战斗内映射 |
|----------|------------|
| physique | 气血、韧性、护体效率 |
| combat | 招式基础伤害、命中、招架 |
| wisdom | 洞察、阵法效率、资源控制 |
| stealth | 先手、奇袭、脱离 |
| survival | 地形适应、持续恢复 |
| magic | 术法强度、阵眼效率 |
| social | 威慑、对峙、特殊剧情战斗分支 |
| charm | 特殊魅惑/扰心/人心战效果 |

### 3.12.4 建议的伤害公式

我们的伤害建议采用以下结构：

**最终伤害 =（招式基础值 + 武意加成 + 专属资源加成 + 状态修正）× 技能倍率 - 目标减伤 - 护体吸收**

特点：

- 先加法，后乘法
- 防御放在后段结算
- 护体单独表现
- 多段技能逐段独立结算

这样能保留：

- 多段 vs 单段差异
- 破绽、护体、罡劲的清晰价值
- 兵器与心法的插入空间

### 3.12.5 暴击 / 闪避 / 反制

不建议做高随机暴击系统，而建议做：

- **会心**：当命中远高于对方招架时，额外+1层破绽或额外段
- **闪位**：由残影/轻功触发，直接免掉一次非锁定攻击
- **反制**：架势成功时返还 1 点柔/刚武意并打断对方相承

这样更像武侠“斗招”，而不是西式 RPG 赌暴击。

### 3.12.6 斩杀机制

刀法和部分剑道绝学建议拥有“斩杀阈值”：

- 敌方气血低于 X%
- 且已存在破绽/失衡/断脉之一
- 则绝学追加斩杀段

这能强化名场面感。

## 3.13 与现有系统的接口设计

## 3.13.1 与卡牌系统的接口

当前项目已有：

- 人物卡
- 装备卡
- 消耗品卡
- 情报卡

战斗系统可直接复用：

### 人物卡

- 继续作为战斗单位
- 现有 8 维属性作为战斗二级数值来源
- 角色标签决定武学适配度

### 装备卡

- 武器：决定兵器技与主武学倾向
- 防具：提高气血/护体/罡劲
- 饰品：提高武意权重、清状态、改绝学阈值

### 情报卡

- 战前窥探敌方招式
- 直接赋予“首回合洞察”
- 或在 Boss 战中揭露“破解机制”

### 消耗品卡

- 提供临时武意
- 提供清状态
- 提供一次性身法/绝学增强

## 3.13.2 与场景系统的接口

建议在现有 `settlement` 体系上新增一种设计概念：

- **battle settlement**

其本质仍可输出：

- success
- partial_success
- failure
- critical_failure

与当前 `effects` 完整兼容。

战斗在场景层面可作为：

1. 单阶段纯战斗
2. 多阶段事件中的一个分支
3. Boss 场景的核心结算段

### 场景地形的战斗价值

场景天然适合提供战斗 modifier：

- 雪地：重 + 柔收益上升
- 城巷：奇 + 疾收益上升
- 山巅：玄系阵法收益上升
- 马战平原：枪法和冲锋收益上升

这让场景系统不只是“皮肤”，而是战斗环境。

## 3.13.3 与事件系统的接口

事件节点可以做三类战斗前置：

1. **情绪前置**
   - 如是否激怒对手，决定 Boss 开场状态

2. **资源前置**
   - 如是否带到某件兵器/援护人物

3. **叙事分支前置**
   - 如“单挑”“群战”“护送”“夜袭”决定战斗目标不同

## 3.13.4 与当前检定系统的接口

战斗不应取代检定，而应成为检定的升级层。

建议规则：

- 小冲突仍用骰检
- 重要冲突进入战斗
- 战前骰检可为战斗提供前置优势

例如：

- 社交检定成功：首回合敌方带 2 层破绽
- 情报检定成功：揭露敌方绝学冷却
- 隐匿检定成功：我方夺先手

这样保留当前项目已有的骰检价值。

## 3.15 为什么这套方案适合我们当前项目

## 3.15.1 它尊重现有工程结构

当前项目已经有：

- 人物卡
- 装备卡
- 场景
- 事件阶段
- 结算分支

这套战斗方案不是另起炉灶，而是：

> **把“重要冲突”从单次骰检升级为可构筑、可叙事、可角色化的战斗结算模块。**

## 3.15.2 它尊重 IP

《雪中悍刀行》的核心不是门派大全，而是：

- 人物风采
- 招式气象
- 临战关系
- 名场面瞬间

这套设计的重心正好落在：

- 人物专属机制
- 角色配合
- 招式意境
- 战斗转折

## 3.15.3 它吸收了《觅长生》的长处，也避开了它的短处

保留：

- 资源编排
- 顺序连携
- 状态叠层
- 流派闭环
- 构筑反制

规避：

- 过长的单回合操作
- 过慢的普通战节奏
- 过重的系统理解负担

## 4. 最终设计结论

如果用一句话总结这套方案：

> **以“武意轮 + 专属资源 + 角色名招 + 场景地形 + 事件前置”构成一套偏策略、强人物、快节奏的武侠回合战斗系统。**

它对《觅长生》的真正继承，不是五行本身，而是以下五个原则：

1. 资源要有波动，但能被构筑驯服
2. 顺序要有价值，出招要讲招路
3. 被动要能形成引擎
4. 状态要能撑起流派
5. 敌人要用机制克制玩家，而不是只堆数值

而它对《雪中悍刀行》的真正还原，是把人物写进规则里：

- 徐凤年的借势成刀
- 李淳罡的剑开天门
- 徐龙象的金刚压城
- 洪洗象的天象阵机
- 红薯的绵里藏针
- 老黄的藏锋九剑

这样，战斗就不只是“谁数值高谁赢”，而是“**谁把江湖打出来了**”。

## 5. 附录：本次研究参考要点

### 《觅长生》研究重点依据

- 战斗为卡牌感回合制，核心资源为五行灵气，且有保留上限
- 灵根与权重影响每回合灵气分布
- 主动技能主要来自神通/武器，被动主要来自功法
- 五行顺序能产生相承/相克式收益
- 功法存在开局触发、回合触发、出手触发、顺序触发、资源回流等模式
- 状态系统高度离散化，护罩/霜冻/缠绕/中毒/灼烧/易伤等都可明确堆层
- 高级流派以“引擎闭环”而非“单卡强度”为核心
- 敌人和 Boss 会以机制压制特定 build

### 《雪中悍刀行》转译重点依据

- 李淳罡：剑道魁首，名招适合高阈值爆发与破势
- 徐凤年：杂糅百家、善借势，不应设计成单一兵器模板
- 徐龙象：金刚体魄、压阵型角色，天然适合高韧高压前排
- 洪洗象：道心通明，适合阵法、天象、延时型规则改写
- 红薯：智谋、洞察、柔中藏锋，适合奇策与辅控
- 老黄：藏锋护主、后手翻盘，适合保护与蓄剑资源体系
