# 《雪中悍刀行》属性命名升级方案

## 0. 文档目标

本文档聚焦 **“命名升级”**，不改动现有属性机制、不改动数值逻辑、不改动底层字段键名，仅针对当前 `sutan` 项目中的人物属性、装备属性、物品属性，提出一套更贴近《雪中悍刀行》武侠语境的显示命名方案。

目标约束如下：

1. **保留现有结构不变**
   - 角色仍然是 8 维基础属性
   - 特殊属性仍然是 `support / reroll`
   - 装备、情报、思绪、宝石仍通过 `attribute_bonus / special_bonus` 生效
   - 场景判定仍使用同一套属性键
2. **只升级命名，不升级规则**
   - 不修改枚举
   - 不修改 schema
   - 不修改数值范围
   - 不修改判定公式
3. **对标《雪中悍刀行》**
   - 命名需要更有江湖、庙堂、武道、气机、道门术法的气质
   - 同时必须让玩家一眼能理解，不走生僻玄学路线

## 1. 现有属性系统梳理

## 1.1 系统结构总览

当前项目的属性系统可以分为三层：

### A. 角色基础属性层

- 数据字段：`attributes`
- 适用对象：`character`
- 结构：固定 8 维
- 用途：
  - 角色卡核心强度表达
  - 场景 `dice_check` 的基础骰池来源
  - 装备加成的承接对象

### B. 角色/装备特殊属性层

- 角色字段：`special_attributes`
- 物品字段：`special_bonus`
- 当前字段：
  - `support`
  - `reroll`
- 用途：
  - 作为额外战术能力显示
  - `reroll` 已实际进入骰子结算
  - `support` 当前主要作为展示与数据字段存在，文义上代表协助/辅助能力

### C. 物品效果属性层

- 数据字段：`attribute_bonus`
- 适用对象：
  - `equipment`
  - `intel`
  - `gem`
  - `thought`
- 用途：
  - 作为投入物品或已装备物品时，对某一基础属性提供加值
  - 装备会叠加到角色总属性
  - 非角色投入卡会直接增加场景判定骰池

| `social` | 社交 / 交际 | 人际往来、口才、关系经营 | 角色基础属性；用于周旋、说服、场面经营；可被物品加成 |

| `magic` | 魔力 / 法术 / 秘术 / 灵力 | 术法、修行、玄门能力 | 角色基础属性；用于玄学/术法型角色；可被装备加成 |

补充说明：

- 这 8 个键名已经贯穿：
  - 基础卡牌数据
  - 场景判定配置
  - schema 校验
  - 装备加值
  - 素材管理器
- 因此本次设计建议只改 **中文显示名**，不改内部 key。

## 1.6 当前属性在系统中的实际使用方式

## 1.6.1 场景判定

场景的 `dice_check` 使用：

- `attribute`
- `calc_mode`
- `target`

判定流程是：

1. 从投入的角色中按 `attribute + calc_mode` 计算基础骰池
2. 累加该角色已装备物品的 `attribute_bonus`
3. 若还投入了非角色物品卡，再叠加其对应 `attribute_bonus`
4. 累加角色与装备带来的 `reroll`
5. 进入掷骰结算

这意味着：

- 8 维属性既是“角色画像”，也是“判定入口”
- 装备/情报/思绪等物品本质上是对同一属性语言体系的修正器
- 命名一旦升级，必须保证“判定名”和“物品加成名”仍然能无缝对齐

## 1.6.2 装备系统

装备系统的真实效果是：

- `attribute_bonus`：叠加到角色总属性
- `special_bonus.reroll`：叠加可重掷次数
- `special_bonus.support`：作为特殊属性保留

所以装备属性名必须让玩家理解：

- 这是“加在哪个能力维度上”
- 同一个属性名必须同时适用于人物面板、场景判定、装备词条

## 2. 对标《雪中悍刀行》世界观的命名适配性分析

## 2.1 《雪中悍刀行》的气质关键词

若对标《雪中悍刀行》，属性名需要体现以下四类语境：

1. **江湖语境**
   - 刀剑
   - 身法
   - 气势
   - 风骨
   - 游历
   - 人情
2. **庙堂语境**
   - 权谋
   - 察势
   - 周旋
   - 名望
   - 手腕
3. **武道语境**
   - 筋骨
   - 武艺
   - 架势
   - 气机
   - 杀力
4. **玄门/修行语境**
   - 道心
   - 术法
   - 感应
   - 修为
   - 玄机

因此，太偏西幻、太偏现代社交、太偏通用 RPG 的命名，会削弱《雪中悍刀行》的辨识度。

---

## 2.2 现有命名的整体问题

当前命名存在四个层次的问题：

### A. 风格不统一

同一套属性在不同位置出现不同译法：

- `combat` 有“战斗 / 武力 / 武艺”
- `wisdom` 有“智慧 / 智识 / 智谋”
- `magic` 有“魔力 / 法术 / 秘术 / 灵力”
- `social` 有“社交 / 交际”

这会导致：

- 玩家不确定这些是不是同一属性
- UI 与工具链对世界观的语气不一致

### B. 部分词偏现代/偏系统化

例如：

- `社交`
- `交际`
- `支持`
- `重投`
- `魔力`

这些词要么偏现代产品语，要么偏桌游术语，要么偏西幻，不够“雪中”。

### C. 部分词偏通用 RPG

例如：

- `战斗`
- `生存`
- `智慧`

意思没错，但过于抽象、过于泛化，没有把人物放进“武侠江湖”的语境里。

### D. 少数词已经比较贴切

例如：

- `体魄`
- `武艺`
- `潜行`
- `秘术`

这些词比“体力 / 攻击 / 隐身 / 魔法”更有武侠味，已经接近目标方向。

---

## 2.3 单项评估

### 1. `physique`

- 当前常见名：体魄
- 评价：**贴切**
- 原因：
  - “体魄”天然带筋骨、气血、耐受意味
  - 与《雪中悍刀行》里“金刚体魄”“熬炼体魄”语感一致
  - 可同时覆盖角色面板、装备词条、场景判定

结论：**不建议改**

### 2. `charm`

- 当前常见名：魅力
- 评价：**基本贴切，但略偏通用**
- 原因：
  - “魅力”玩家理解成本最低
  - 但稍偏现代人格标签
  - 在《雪中》里，许多角色真正起作用的不只是“好看”，更是“风仪”“气度”“摄人心魄”

结论：**建议升级，但不要改得太晦涩**

### 3. `wisdom`

- 当前常见名：智慧 / 智识 / 智谋
- 评价：**需要统一，且应更偏谋局**
- 原因：
  - 在本作里，它不仅是聪明，更包含判断、谋略、识势
  - 《雪中》里的强者经常不是“聪明”，而是“心有丘壑、善谋全局”

结论：**建议统一为更有谋局感的名称**

### 4. `combat`

- 当前常见名：战斗 / 武力 / 武艺
- 评价：**“战斗”太游戏化，“武艺”最佳**
- 原因：
  - “战斗”像系统标签，不像江湖词
  - “武力”偏数值暴力，缺少技艺感
  - “武艺”兼容刀、剑、枪、拳、阵前厮杀，是武侠语境里最稳的叫法

结论：**建议统一为“武艺”**

### 5. `social`

- 当前常见名：社交 / 交际
- 评价：**明显偏现代**
- 原因：
  - “社交”是现代社会关系词
  - 《雪中》的关系经营更接近“周旋”“应酬”“处世”“人情往来”
  - 又因为该属性还承担口才、关系、斡旋场景，不能只叫“口才”

结论：**建议更名**

### 6. `survival`

- 当前常见名：生存 / 求生
- 评价：**“生存”太系统，“求生”稍好，但还可更武侠**
- 原因：
  - 本属性不仅是活下来，还包含荒野经验、风餐露宿、临场应变、行走江湖的老练
  - 《雪中》的人物更像是“江湖阅历”“行路本事”

结论：**建议更名**

### 7. `stealth`

- 当前常见名：隐匿 / 潜行
- 评价：**都可用，但“潜行”更易懂**
- 原因：
  - “隐匿”更书面
  - “潜行”更符合游戏操作感
  - 若对标武侠，最好让它同时涵盖身法、藏锋、潜入

结论：**建议小幅升级**

### 8. `magic`

- 当前常见名：魔力 / 法术 / 秘术 / 灵力
- 评价：**必须统一，且要去西幻化**
- 原因：
  - 《雪中》不是西幻魔法世界
  - 相关能力更接近术法、玄门、气机感应、修为层面的“术”
  - “魔力”是最不贴世界观的说法

结论：**必须更名**

### 9. `support`

- 当前常见名：支持 / 支援
- 评价：**词义正确，但气质太弱**
- 原因：
  - “支持”非常像系统按钮
  - “支援”稍好，但仍偏现代战术词
  - 《雪中》语境里，更接近“策应”“照应”“掠阵”

结论：**建议更名**

### 10. `reroll`

- 当前常见名：重投 / 重骰 / 重掷
- 评价：**机制词明确，但表现过于桌游化**
- 原因：
  - 玩家能理解，但美术与文案语气会很出戏
  - 若只做显示层升级，最好把它包装成“临阵应变/变招机会”

结论：**建议改为更有武侠临场感的名称**

## 3. 命名升级方案

## 3.1 命名原则

本次升级遵循 5 条原则：

1. **内部键名不动，中文显示升级**
2. **先保证易懂，再追求风味**
3. **人物属性、装备属性、场景判定必须共用同一套叫法**
4. **优先采用《雪中悍刀行》可自然成立的语义**
5. **不硬塞过深典故，避免“看起来高级但玩家看不懂”**

| `charm` | 魅力 | **风仪** | 是 | 比“魅力”更武侠，兼顾容貌、气质、风度 |

| `social` | 社交 / 交际 | **周旋** | 是 | 兼容口才、人情、应对、关系经营 |

## 3.3 各名称详细论证

### `physique` → 体魄（保留）

保留原因：

- 这是现有体系里最好的名称之一
- 能同时承载：
  - 筋骨
  - 气血
  - 抗打
  - 体力
- 也与《雪中》“金刚体魄”“横练筋骨”的语义天然相合

不建议替换成：

- 根骨：更像天赋资质，不等于当前实际用途
- 筋骨：太偏身体组织，不够完整
- 气血：偏状态，不够泛用

### `charm` → 风仪

推荐理由：

- “风仪”比“魅力”更有古风和门阀气
- 能覆盖：
  - 容貌
  - 气度
  - 风采
  - 令人折服或心动的存在感
- 《雪中》中的徐脂虎、南宫仆射、轩辕青锋，并不是简单“魅力高”，而是“风仪出众”

风险：

- 对少数玩家来说，第一眼不如“魅力”直白

结论：

- 若追求世界观氛围，**风仪是最佳升级项**
- 若追求极限易懂，也可保守使用“魅力”

### `wisdom` → 谋略

推荐理由：

- 当前属性用途不仅是智商，更是谋局与识势
- “谋略”能直接把人物拉进《雪中》的庙堂和江湖大局里
- 像徐凤年、徐骁、李义山、陈芝豹、轩辕敬城，这一维明显更接近“谋略”而非“智慧”

不建议继续混用：

- 智慧：太泛
- 智识：更偏知识与见闻
- 智谋：也可用，但“谋略”更稳、更中性

### `combat` → 武艺

推荐理由：

- 当前最适合统一的名字
- “武艺”比“战斗”更有人味，也比“武力”更有技法感
- 可兼容刀剑、枪术、拳脚、马上战、杀伐经验

结论：

- 这是本次最确定的一项统一升级

### `social` → 周旋

推荐理由：

- 《雪中》的关系经营不是现代意义的“社交”
- 更像：
  - 察言观色
  - 应酬往来
  - 斡旋关系
  - 场面拿捏
  - 利益折冲
- “周旋”恰好能同时覆盖人情与权谋

备选：

- 人情：有味道，但太窄，不含谈判
- 处世：太软，不够动态
- 斡旋：稍显偏书面

综合结论：

- **周旋**是最平衡的方案

### `survival` → 江湖

推荐理由：

- 这是本次最有风味、也最值得升级的一个名称
- 当前 `survival` 的实际含义，不只是求生技能，而是：
  - 风餐露宿
  - 行路经验
  - 危机应对
  - 吃苦耐受
  - 对复杂环境的适配能力
- 在《雪中》里，这类能力最像“江湖经验”

风险：

- “江湖”不是纯能力名，而是氛围型命名
- 需要在 UI 或引导文案中让玩家自然理解其代表“行走江湖的本事”

备选：

- 求生：更直白，但风味弱
- 历练：更像成长结果
- 行藏：太文

综合结论：

- 如果项目愿意接受更有味道的命名，**江湖**非常适合

### `stealth` → 身法

推荐理由：

- 在武侠里，隐蔽行动通常不只是“藏起来”
- 它常常包括：
  - 轻功
  - 近身机动
  - 藏锋敛迹
  - 潜入突袭
- “身法”能比“潜行”承载更多武侠动作意味

风险：

- “身法”会比“潜行”略微偏宽，包含闪避和步法联想

结论：

- 从武侠氛围看，**身法**优于“潜行/隐匿”

### `magic` → 术法

推荐理由：

- 《雪中》里不适合出现“魔力”这种西幻词
- “术法”兼容：
  - 道门法门
  - 玄门术数
  - 气机牵引
  - 修为外显
- 洪洗象、轩辕敬城、寒玉戒指、十二飞剑，都更像“术法”相关，而不是“魔法”

备选：

- 秘术：更神秘，但略偏具体招数
- 修为：过大，像总境界
- 气机：很雪中，但不够直观，且与体魄/武艺关系也密切

综合结论：

- **术法**是最佳主案

### `support` → 策应

推荐理由：

- “策应”比“支持/支援”更像江湖并肩、战阵呼应
- 能覆盖：
  - 照拂队友
  - 从旁成局
  - 帮人接招
  - 压阵

备选：

- 掠阵：风味很强，但更偏战斗
- 照应：生活味重，战斗感弱
- 援手：太口语

综合结论：

- **策应**最均衡

### `reroll` → 变招

推荐理由：

- 现有机制本质是“失败后再调整一次出手”
- 用“变招”包装后，玩家很容易接受：
  - 临场反应
  - 改手
  - 二次应对
  - 见招拆招
- 非常符合武侠战斗叙述习惯

风险：

- 这是“表现型翻译”，不是严格机制直译

结论：

- 如果目标是世界观沉浸，**变招**比“重掷/重骰”更好

如果团队担心“风仪 / 周旋 / 江湖 / 身法 / 变招”过于风味化，可以采用一套更稳妥的中间方案：

| 内部键名 | 保守版名称 |
|---|---|
| `physique` | 体魄 |
| `charm` | 魅力 |
| `wisdom` | 谋略 |
| `combat` | 武艺 |
| `social` | 人情 |
| `survival` | 求生 |
| `stealth` | 潜行 |
| `magic` | 术法 |
| `support` | 策应 |
| `reroll` | 变招 |

这套方案的特点是：

- 比现状更统一
- 比主案更易懂
- 风味略弱，但落地风险更低

## 3.5 最终推荐

综合世界观贴合度、玩家理解成本、与现有数值语义的一致性，本文推荐：

### 最终推荐主案

| 内部键名 | 最终推荐中文名 |
|---|---|
| `physique` | 体魄 |
| `charm` | 风仪 |
| `wisdom` | 谋略 |
| `combat` | 武艺 |
| `social` | 周旋 |
| `survival` | 江湖 |
| `stealth` | 身法 |
| `magic` | 术法 |
| `support` | 策应 |

   - 江湖、庙堂、谋局、术法、身法等关键词都被覆盖

## 4.2 装备与物品词条

装备、情报、宝石、思绪的 `attribute_bonus`、`special_bonus`，建议沿用同一显示名。

例如：

- `combat +7` → `武艺 +7`
- `social +3` → `周旋 +3`
- `magic +5` → `术法 +5`
- `reroll +1` → `变招 +1`

这样玩家更容易建立直觉：

- 人物有一套能力名
- 装备只是给这套能力名加值
- 场景也用这套能力名做判定

## 4.3 场景判定文本

场景文本中当前常见写法如：

- 进行「魅力」鉴定
- 进行「社交」鉴定
- 进行「智慧」鉴定

如果采用主案，则应升级为：

- 进行「风仪」鉴定
- 进行「周旋」鉴定
- 进行「谋略」鉴定
- 进行「术法」鉴定

收益：

- 判定文案更像武侠叙事
- 玩家会更容易把“属性”理解为人物气质的一部分，而不是纯数值面板

## 7. 风险与建议

## 7.1 风险点

### 风险 1：风味增强后理解门槛上升

例如：

- `风仪` 不如 `魅力` 直白
- `江湖` 不如 `求生` 直白
- `变招` 不如 `重掷` 直白

### 风险 2：局部更新会导致命名不一致

若只改主 UI，不改：

- 资产管理器
- 场景生成器
- Prompt

则团队内部使用和玩家外部看到的语言会不统一。

### 风险 3：个别名称有“氛围翻译”成分

尤其是：

- `survival → 江湖`
- `reroll → 变招`

它们更像“世界观表现名”，不是字面直译。

---

## 7.2 建议落地策略

建议分两步：

### 方案 A：一次到位主案

适合目标：

- 强化《雪中悍刀行》辨识度
- 愿意用更强世界观语言包装系统

采用命名：

- 体魄 / 风仪 / 谋略 / 武艺 / 周旋 / 江湖 / 身法 / 术法 / 策应 / 变招

### 方案 B：稳妥过渡版

适合目标：

- 先统一命名，再逐步增强风味

采用命名：

- 体魄 / 魅力 / 谋略 / 武艺 / 周旋 / 求生 / 潜行 / 术法 / 策应 / 变招

本文更推荐 **方案 A**，因为它更符合当前任务“对标《雪中悍刀行》”的初衷。

## 8. 结论

从当前系统看，`sutan` 的属性结构本身已经足够稳：

- 8 维基础属性负责人物画像与判定入口
- `support / reroll` 负责特殊能力表达
- 装备/物品通过同名 bonus 体系叠加

问题不在机制，而在 **命名风格尚未完全进入《雪中悍刀行》的语境**。

因此，本次最合适的升级路径不是重做系统，而是：

> **保留底层键名与逻辑，统一所有显示层命名，把通用 RPG 词汇升级为更有江湖、庙堂、术法气质的武侠词汇。**

最终建议采用以下命名：

| 内部键名 | 建议显示名 |
|---|---|
| `physique` | 体魄 |
| `charm` | 风仪 |
| `wisdom` | 谋略 |
| `combat` | 武艺 |
| `social` | 周旋 |
| `survival` | 江湖 |
| `stealth` | 身法 |
| `magic` | 术法 |
| `support` | 策应 |
| `reroll` | 变招 |

这套命名的价值在于：

1. 不动底层结构，实施成本低
2. 风格统一，玩家认知更稳定
3. 更贴《雪中悍刀行》的人物气质与江湖表达
4. 能直接服务后续角色、装备、场景文案的整体风格升级
