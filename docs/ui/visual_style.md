# 视觉风格

本文档定义游戏的整体美术风格与视觉规范。

## 整体美术风格

- **主题**：阿拉伯/中东风格 + 暗黑奇幻 + 蒸汽朋克
- **配色**：深红褐色主色调 + 金色装饰 + 多彩卡牌
- **质感**：羊皮纸纹理、深色皮革、黄铜金属

---

## 卡牌设计风格

- 竖长型塔罗牌比例
- 手绘线稿或淡彩插图
- 破损做旧边框效果
- 神秘学/克苏鲁元素

### 稀有度视觉

| 稀有度 | 边框/背景色 |
|--------|-------------|
| 金（gold） | 金色 |
| 银（silver） | 银色 |
| 铜（copper） | 铜色 |
| 石（stone） | 灰色/石质 |

---

## UI设计原则

- **信息层级清晰**：重要信息突出显示
- **操作反馈明确**：各功能区有明确图标和说明
- **引导友好**：提供教程系统和帮助按钮
- **容错机制**：时间回溯功能允许玩家纠正错误

---

## 色彩规范

### 主色调

| 用途 | 色彩 |
|------|------|
| 背景 | 深红褐色 |
| 装饰边框 | 金色 |
| 文字 | 米白/羊皮纸色 |
| 高亮 | 亮金色 |

### 功能色

| 用途 | 色彩 |
|------|------|
| 必选/危险 | 红色 |
| 可选/安全 | 白色/浅色 |
| 成功 | 绿色/金色 |
| 失败 | 暗红色 |

---

## 质感规范

### 背景质感

- **深色皮革**：主界面背景
- **羊皮纸**：文本区域、对话框
- **黄铜金属**：边框、装饰元素

### 边框风格

- **蒸汽朋克**：齿轮装饰、金属铆钉
- **古典花纹**：四角装饰
- **做旧效果**：磨损、斑驳

---

## 图标风格

- 线条简洁
- 金色或单色
- 符合中东/阿拉伯风格
- 神秘学元素（星象、符文等）

---

## 动效规范

### 卡牌动效

- 悬停：轻微放大、边框高亮
- 拖拽：跟随鼠标、半透明
- 放置：落位动画、闪光反馈

### 骰子动效

- 投掷：3D翻滚
- 结果：数字闪烁、成功/失败色彩
- 爆骰：额外骰弹出动画

### 界面切换

- 淡入淡出
- 滑动过渡
- 避免生硬跳转
